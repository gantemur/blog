Дөрвөн зэргийн тэгшитгэлийг бодох ерөнхий аргыг анх олсон хүн бол [Жероламо Карданы](https://en.wikipedia.org/wiki/Gerolamo_Cardano) авъяаслаг шавь [Лодовико Феррари](https://en.wikipedia.org/wiki/Lodovico_Ferrari) (1522–1565) юм. Тэрбээр 14 настайдаа Карданы шавь болсон ба авъяас билэг нь тэр дороо цэцэглэж, Кардантай мөр зэрэгцэн судалгаа хийж эхэлсэн. Ингээд 20 хүрээгүй байхдаа Ром хотод математикийн профессор болж, төд удалгүй Милан хотын захирагчийн дэргэд цалин ихтэй албанд ороод, 42 настайдаа нилээд хөрөнгөжүү хүн тэтгэвэртээ гарсан боловч, жилийн дараа хэн нэгэнд хорлогдон нас баржээ. Карданы бичсэнээр бол Феррариг дүү нь хорлосон гэдэг. Ямар ч байсан Маддалена гэгдэх дүү нь Феррарийн бүх хөрөнгийг өвлөж авсан ба Феррарийн оршуулганд ч оролцоогүйгээр барахгүй оршуулганаас нь 14 хоногийн дараа нөхөрт гарсан гэж байгаа. Гэхдээ шинэ нөхөр нь удалгүй бүх хөрөнгийг нь аваад салж, Маддалена өөрөө нас эцэслэтлээ гуйлгачны амьдралаар амьдарсан гэдэг.

Ферраригаас өмнө кубээс дээш зэргийн тэгшитгэлийн судалгаа нь зөвхөн дараах чиглэлүүд дээр төвлөрч байв: 

 
-  олон үл мэдэгдэгчтэй тохиолдол, 
-  квадрат тэгшитгэлд шууд шилжих (биквадрат гэх мэт) тэгшитгэлүүд, 
-  шийдийг орон орноор нь тооцдог ойролцоо аргууд. 

Тухайлбал, агуу Диофант өөрийн «[Арифметик](https://en.wikipedia.org/wiki/Arithmetica)» (МЭ 250 оны орчим) бүтээлд олон үл мэдэгдэгчтэй (3, 4, 6 зэргийн) олон гишүүнт тэгшитгэлийн (болон систем тэгшитгэлийн) рациональ шийдүүдийг судалсан.

Цаашилбал, эртний Хятадад [Ван Сяо-тун](https://en.wikipedia.org/wiki/Wang_Xiaotong) (МЭ 580–640) биквадрат буюу 

\(\displaystyle  x^4+bx^2=c\)

 хэлбэрийн тэгшитгэлүүдийг бодох аргыг олсон бол, Энэтхэгт [Махавира](https://en.wikipedia.org/wiki/Mahavira_(mathematician)) (МЭ 8-р зуун) 

\(\displaystyle  a(x^n-1)=b(x-1)\)

 болон 

\(\displaystyle  a\sqrt{x}+b\sqrt{x-a\sqrt{x}}=c\)

 маягийн тэгшитгэлүүдийн шийдийг шууд тааж олсон байдаг. Түүнчлэн, Исламын эзэнт гүрний [аль-Каражи](https://en.wikipedia.org/wiki/Al-Karaji) (953–1029) ба [аль-Бируни](https://en.wikipedia.org/wiki/Al-Biruni) (973–1050) нар 

\(\displaystyle  x^{2n+m}+bx^{n+m}=cx^m\)

 хэлбэрийн тэгшитгэлийг бодож чаддаг байсан. Мөн [Ибн Лайс](https://en.wikipedia.org/wiki/Abu_al-Jud) (10-р зуун) зарим куб болон 4 зэрэгт тэгшитгэлийг конус огтлол ашиглан бодсон. Нөгөө талаас, [Аль-Бируни](https://en.wikipedia.org/wiki/Al-Biruni) (973–1050), [Омар Хайям](https://en.wikipedia.org/wiki/Omar_Khayyam) (1048–1131), [Мароккийн Самуил](https://en.wikipedia.org/wiki/Al-Samawal_al-Maghribi) (1130–1180) нар тооноос \({n}\) зэргийн язгуурыг орон орноор нь гаргах алгоритмыг боловсруулжээ. Үүнтэй төстэй алгоритмыг Хятадад [Цзя Сянь](https://en.wikipedia.org/wiki/Jia_Xian) (1010–1070) куб, 4 зэргийн язгуурын хувьд нээсэн нь 200-аад жилийн дотор дурын зэргийн олон гишүүнт тэгшитгэлийн *системийг* бодох алгоритм болтлоо хөгжсөн. Тухайлбал, [Цинь Цзюшао](https://en.wikipedia.org/wiki/Qin_Jiushao) (1202–1261) «[Математикийн 9 ном](https://en.wikipedia.org/wiki/Mathematical_Treatise_in_Nine_Sections)» (1247) бүтээлдээ 10 хүртэлх зэргийн тэгшитгэлийг, [Чжу Шицзе](https://en.wikipedia.org/wiki/Zhu_Shijie) (1249–1314) «[Дөрвөн үл мэдэгдэгчийн хаш толь](https://en.wikipedia.org/wiki/Jade_Mirror_of_the_Four_Unknowns)» (1303) бүтээлдээ 14 хүртэлх зэргийн тэгшитгэл төдийгүй 4 ширхэг олон гишүүнт тэгшитгэлээс бүтсэн системийг бодсон байдаг. Харин Арабчуудын хувьд иймэрхүү алгоритмыг [Шарафуддин ат-Туси](https://en.wikipedia.org/wiki/Sharaf_al-Din_al-Tusi) (1135–1213) куб тэгшитгэлийн хувьд боловсруулсан боловч кубээс дээш зэргийн олон гишүүнтүүд рүү өргөтгөөгүй.

Эцэст нь тэмдэглэхэд, Энэтхэгийн [Бхаскара II](https://en.wikipedia.org/wiki/Bhāskara_II) (1114–1185) 

\(\displaystyle  x^4-2x^2-400x=9999\)

 тэгшитгэлийг \({x^4+2x^2+1=4x^2+400x+10000}\) хэлбэрт шилжүүлээд, хоёр талаас нь шууд язгуур авах замаар бодож байсан. 

##  Феррари 

Феррарийн аргыг [Карданы номын](https://en.wikipedia.org/wiki/Ars_Magna_(Gerolamo_Cardano)) нэг жишээгээр эхлээд танилцуулъя. Кардано, Феррари нарын авч үзсэн тэгшитгэл нь [

\(\displaystyle  x^4=x+2 . \ \ \ \ \ (1)\)

] Энэ тэгшитгэлийн зүүн гар тал бүтэн квадрат хэвээрээ үлдэхээр, \({2tx^2+t^2}\) гэсэн илэрхийллийг хоёр тал дээр нь нэмбэл [

\(\displaystyle  x^4 + 2tx^2 + t^2 = 2tx^2 + x + 2 + t^2 \ \ \ \ \ (2)\)

] болно. Бидний дараагийн зорилго бол энэ тэнцэтгэлийн баруун гар талыг бүтэн квадрат байхаар \({t}\) хувьсагчийн утгыг тохируулах явдал юм. Өөрөөр хэлбэл, \({2tx^2 + x + 2 + t^2}\) илэрхийллийг \({x}\)-ийн хувьд квадрат олон гишүүнт гэж үзээд, дискриминантыг нь 0 болгодог тийм \({t}\)-ийн утгыг олно гэсэн үг: [

\(\displaystyle  1 - 8t(2+t^2) = 0 . \ \ \ \ \ (3)\)

] Үүнийг *ресольвент тэгшитгэл* гэж нэрлэдэг бөгөөд, энэ нь *куб тэгшитгэл* тул [Карданы аргаар](/blog/2019/06/22/cubic-equations/) шууд бодогдоно. Дараах илэрхийллийг [(2)](#eferrari-eg) тэнцэтгэлийн баруун гар талтай харьцуулах замаар ресольвент тэгшитгэлийг гаргаж ч болно: 

\(\displaystyle  (x\sqrt{2t}+\sqrt{2+t^2})^2 = 2tx^2 + 2x\sqrt{2t(2+t^2)} + 2 + t^2 .\)

 Ингээд ресольвент тэгшитгэлийн *нэг шийдийг л* олчихвол тэр \({t}\)-гийн утгыг [(2)](#eferrari-eg) тэнцэтгэлд орлуулснаар 

\(\displaystyle  (x^2 + t)^2 = \big( x\sqrt{2t}+\sqrt{2+t^2} \big)^2\)

 буюу 

\(\displaystyle  \big( x^2 + t + x\sqrt{2t} + \sqrt{2+t^2} \big) \big( x^2 + t - x\sqrt{2t} - \sqrt{2+t^2} \big) = 0\)

 болох ба, анхны [(1)](#eferrari-cardano-eg) тэгшитгэл маань 2 ширхэг квадрат тэгшитгэлд задарна. Феррарийн аргын гол цөм нь энэ л юм.

Одоо дээрх аргыг ерөнхий коэффициенттэй 4 зэрэгт тэгшитгэлийн хувьд өргөтгөж бичье. Юуны өмнө, ерөнхий [

\(\displaystyle  z^4+az^3+bz^2+cz+d=0 \ \ \ \ \ (4)\)

] хэлбэрийн тэгшитгэлийг 

\(\displaystyle  z=x-\frac{a}4\)

 орлуулгаар [

\(\displaystyle  x^4+px^2+qx+r=0 \ \ \ \ \ (5)\)

] гэсэн *хураангуй* хэлбэрт шилжүүлж болно. Үүнд 

\(\displaystyle  \begin{array}{rcl}  p &=& b - \frac{3a^2}8 , \\ q &=& c - \frac{ab}2 + \frac{a^3}{8} , \\ r &=& d - \frac{ac}4 + \frac{a^2b}{16} - \frac{3a^4}{256} . \end{array}\)

 Өмнөх жишээнд үзүүлсэн аргыг даган, [(5)](#equart-depressed) тэгшитгэлийн \({x^4}\)-өөс бусад гишүүдийг нь баруун гар талд нь гаргаад, хоёр талд нь \({2tx^2 + t^2}\) илэрхийллийг нэмбэл [

\(\displaystyle  x^4 + 2tx^2 + t^2 = (2t-p)x^2 - qx + t^2-r \ \ \ \ \ (6)\)

] болох ба, үүний баруун гар тал нь \({(kx+m)^2}\) маягийн бүтэн квадрат байх нөхцлийг дискриминант ашиглан олбол бид 

\(\displaystyle  q^2 - 4(2t-p)(t^2-r) = 0 ,\)

 буюу [

\(\displaystyle  \textstyle t^3 - \frac12pt^2 - rt + \frac12pr-\frac18q^2 = 0 . \ \ \ \ \ (7)\)

] гэсэн (Феррарийн) *ресольвент тэгшитгэлд* хүрнэ. Энэ нь \({t}\)-ийн хувьд куб тэгшитгэл бөгөөд, аль ч шийдийнх нь хувьд [(6)](#eferrari) тэнцэтгэл 

\(\displaystyle  (x^2 + t)^2 = \big( x\sqrt{2t-p}+\sqrt{t^2-r} \big)^2 ,\)

 буюу [

\(\displaystyle  \big( x^2 + t + x\sqrt{2t-p} + \sqrt{t^2-r} \big) \big( x^2 + t - x\sqrt{2t-p} - \sqrt{t^2-r} \big) = 0 , \ \ \ \ \ (8)\)

] хэлбэрт орж, [(5)](#equart-depressed) тэгшитгэл маань 2 ширхэг квадрат тэгшитгэлд задрах юм.

**Жишээ 1.** [] Дараах тэгшитгэлийг авч үзье. 

\(\displaystyle  x^4-12x+3=0.\)

 Үүний ресольвент нь 

\(\displaystyle  t^3 - 3t - 18 = 0 ,\)

 бөгөөд, \({t=3}\) гэсэн шийдийг (18-ын бүхэл хуваагчид дотор хайх замаар) тааж олоход амархан. Ингээд [(8)](#equart-ferrari-factor) задаргааг нь шууд бичвэл 

\(\displaystyle  \big( x^2 + 3 + x\sqrt{6} + \sqrt{6} \big) \big( x^2 + 3 - x\sqrt{6} - \sqrt{6} \big) = 0 ,\)

 болно. Ресольвент тэгшитгэл 3 шийдтэй байдаг нь, анхны тэгшитгэлийн 4 шийдийг хоёр хоёроор нь 3 янзаар салгаж болдогт харгалздаг байх гэсэн таамаглал төрнө. 

**Жишээ 2.** [] Дээрх жишээний төгсгөлд төрсөн таамаглалыг шалгах зорилгоор 

\(\displaystyle  x^4-1=0\)

 гэсэн хялбар жишээ авч үзье. Үүний шийдүүд нь мэдээж 

\(\displaystyle  x_{1,2} = \pm 1, \qquad x_{3,4} = \pm i ,\)

 ресольвент нь 

\(\displaystyle  t^3 + t = 0 .\)

 Ресольвентийн 3 шийдэнд харгалзах [(8)](#equart-ferrari-factor) задаргааг жагсааж бичвэл 

\(\displaystyle  \begin{array}{rcl}  t=0 :&\qquad& (x^2+1)(x^2-1) = 0 , \\ t=i :&\qquad& (x^2+i+x\sqrt{2i})(x^2+i-x\sqrt{2i}) = 0 , \\ t=-i :&\qquad& (x^2-i+x\sqrt{-2i})(x^2-i-x\sqrt{-2i}) = 0 . \end{array}\)

 Харин \({x^4=1}\) тэгшитгэлийн шийдүүдийг хоёр хоёроор нь 3 янзаар салгахад ямар задаргаанууд гарахыг сонирхвол 

\(\displaystyle  \begin{array}{rcl}  \big[(x-i)(x+i)\big]\big[(x-1)(x+1)\big] &=& (x^2+1)(x^2-1) , \\ \big[(x+1)(x+i)\big]\big[(x-1)(x-i)\big] &=& (x^2+i+(1+i)x)(x^2+i-(1+i)x) , \\ \big[(x-1)(x+i)\big]\big[(x+1)(x-i)\big] &=& (x^2-i+(i-1)x)(x^2-i-(1+i)x) . \end{array}\)

 Одоо \({\sqrt{2i}=\pm(1+i)}\) ба \({\sqrt{-2i}=\pm(i-1)}\) гэдгээс бидний таамаглал энэ хялбар жишээн дээр үнэн болох нь харагдана. 

##  Декарт 

Ферраригаас хойш 80 жилийн дараа, 4 зэргийн тэгшитгэлийг бодох шинэ аргыг Францын алдарт философич, математикч [Рене Декарт](https://en.wikipedia.org/wiki/René_Descartes) (1596–1650) олж, «[Геометр](https://en.wikipedia.org/wiki/La_Géométrie)» (1637) бүтээлдээ хэвлүүлсэн. Түүний гол санаа нь 4 зэргийн олон гишүүнтийг квадрат олон гишүүнтүүдийн үржвэрт шууд задлах явдал байв. Хураангуй [(5)](#equart-depressed) хэлбэрийн хувьд, энэ нь [

\(\displaystyle  x^4+px^2+qx+r = (x^2+sx+u) (x^2+tx+v) \ \ \ \ \ (9)\)

] байхаар бичнэ гэсэн үг. Үүний хоёр талын коэффициентүүдийг харгалзан тэнцүүлбэл 

\(\displaystyle  \begin{array}{rcl}  0 &=& s + t , \\ p &=& u + v + st ,\\ q &=& sv+tu , \\ r &=& uv , \end{array}\)

 болох ба, эхний тэнцэтгэлээс \({s=-t}\) гэдгийг голын хоёрт орлуулснаар 

\(\displaystyle  p = u + v - t^2 , \qquad q = (u-v)t ,\)

 гарна. Эндээс 

\(\displaystyle  (u+v)^2 = (p+t^2)^2 , \qquad (u-v)^2t^2=q^2,\)

 гэж олоод, \({4uv = (u+v)^2 - (u-v)^2}\) адилтгалыг ашиглан 

\(\displaystyle  4rt^2 = 4uvt^2 = (u+v)^2t^2 - (u-v)^2t^2 = (p+t^2)^2t^2 - q^2\)

 буюу [

\(\displaystyle  t^6 + 2pt^4 + (p^2-4r) t^2 - q^2 = 0 \ \ \ \ \ (10)\)

] гэсэн (Декартын) *ресольвент тэгшитгэлд* хүрнэ. Хэрэв Декартын ресольвент \({t=0}\) гэсэн шийдтэй бол \({q=(u-v)t=0}\) тул, [(9)](#edescartes) тэгшитгэл маань биквадрат тэгшитгэл болоход хүрч, квадрат тэгшитгэлийн томъёогоор шууд бодогдоно. Ерөнхий тохиолдолд, [(10)](#equart-resolvent-descartes) ресольвент нь \({t^2}\)-ын хувьд куб тэгшитгэл тул [Карданы аргаар](/blog/2019/06/22/cubic-equations/) бодогдох ба, \({t\neq0}\) нөхцөлд 

\(\displaystyle  2u = p+t^2 + \frac{q}t , \qquad 2v = p+t^2 - \frac{q}t ,\)

 гэвэл [(9)](#edescartes) олон гишүүнт [

\(\displaystyle  x^4+px^2+qx+r = (x^2-tx+u) (x^2+tx+v) \ \ \ \ \ (11)\)

] гэж задарна. Цаашилбал, хэрэв анхны [(9)](#edescartes) тэгшитгэлийн коэффициентүүд бодит бол, [(10)](#equart-resolvent-descartes) илэрхийллийн зүүн гар тал \({t\rightarrow\pm\infty}\) үед \({+\infty}\) руу явах ба, \({t=0}\) үед \({-q^2\leq0}\) утгыг авах тул, [(10)](#equart-resolvent-descartes) ресольвент ядаж нэг бодит шийдтэй ([Эйлерийн аргумент](http://eulerarchive.maa.org/docs/translations/E170en.pdf)). Тэгэхээр бодит коэффициенттэй 4 зэргийн олон гишүүнт бүр бодит коэффициенттэй квадратлаг гурван гишүүнтүүдийн үржвэрт задарна гэсэн үг.

**Жишээ 3.** [] Өмнө авч үзсэн 

\(\displaystyle  x^4-12x+3=0\)

 тэгшитгэлд Декартын аргыг хэрэглэе. Декартын ресольвентийг нь бодвол 

\(\displaystyle  \textstyle t^6 - 12t^2 - 144 = 0 ,\)

 бөгөөд, \({t^2=6}\) гэсэн шийдийг (144-ийн бүхэл хуваагчид дотор хайх замаар) тааж олоход амархан. Ингээд \({t=\sqrt6}\) гэж авбал 

\(\displaystyle  u = 3 - \sqrt6, \qquad v = 3 + \sqrt6 ,\)

 гарах ба, [(11)](#equart-descartes-factor) задаргаа нь 

\(\displaystyle  x^4-12x+3 = \big( x^2 - x\sqrt{6} + 3 - \sqrt{6} \big) \big( x^2 + x\sqrt{6} + 3 + \sqrt{6} \big) ,\)

 болно. Хэрэв бид \({t=-\sqrt6}\) гэж авсан бол \({u}\) ба \({v}\)-ийн утгууд хоорондоо солигдох тул, энэ задаргаанд хоёр үржигдэхүүний байр л солигдох байсан. Дээрх задаргаа Жишээ [1](#egferrari-1)-ийн задаргаатай адилхан байгааг анхаар. 

**Дасгал 4.**  Дараах тэгшитгэлийн Декартын ресольвентын бүх шийдийг олоод, шийд тус бүрт нь харгалзах [(11)](#equart-descartes-factor) задаргааг байгуул. 

\(\displaystyle  x^4-1=0 .\)

 

##  Конус огтлол 

МЭ 10-р зуунд Исламын эрдэмтэн [Ибн Лайс](https://en.wikipedia.org/wiki/Abu_al-Jud) зарим куб болон 4 зэргийн тэгшитгэлийг конус огтлол ашиглан бодох аргыг боловсруулсан бол, үүнийг нь сүүлд [Омар Хайям](https://en.wikipedia.org/wiki/Omar_Khayyam) (эерэг шийдтэй) бүх [куб тэгшитгэлийн](/blog/2019/06/22/cubic-equations/) хувьд төгөлдөржүүлжээ. Одоо бид бодит коэффициенттэй бүх куб ба 4 зэргийн тэгшитгэлийг (гортиг шугамаас гадна) *ганц парабол* ашиглан яаж бодож болох байсныг үзүүлье. Юуны түрүүнд, 4 зэргийн ерөнхий тэгшитгэлийн оронд [

\(\displaystyle  x^4+p x^2+q x + r = 0 \ \ \ \ \ (12)\)

] гэсэн *хураангуй* хэлбэрийг нь авч үзэхэд хангалттай гэдгийг санацгаая. Түүнчлэн, куб тэгшитгэлийн 

\(\displaystyle  x^3 + px + q = 0\)

 гэсэн хураангуй хэлбэрийг \({x}\)-ээр үржүүлснээр 

\(\displaystyle  x^4 + px^2 + qx = 0\)

 болох тул, [(12)](#ereduced-quart-khayyam) тэгшитгэлийн \({r=0}\) байх тухайн тохиолдол нь куб тэгшитгэлүүдийг агуулна. Тэгэхээр [(12)](#ereduced-quart-khayyam) тэгшитгэл дээр төвлөрөхөд алдах юм байхгүй.

Одоо 

\(\displaystyle  y = x^2\)

 гэсэн шинэ хувьсагч оруулаад, [(12)](#ereduced-quart-khayyam) тэгшитгэлээ 

\(\displaystyle  y^2+(p-1)y + x^2+q x+r = 0\)

 гэж бичье. Үүний зүүн гар талд \({x}\) ба \({y}\)-ээр бүтэн квадрат ялгавал [

\(\displaystyle  \Big(y+\frac{p-1}2\Big)^2 + \Big(x+\frac{q}2\Big)^2 = \frac{(p-1)^2}4 + \frac{q^2}4 - r \ \ \ \ \ (13)\)

] болно. Эндээс, \({A=(-\frac{p-1}2,-\frac{q}2)}\) цэг дээр төвтэй, \({R=\sqrt{\frac{(p-1)^2}4 + \frac{q^2}4 - r}}\) радиустай тойргийн \({y=x^2}\) параболтой огтлолцох цэгүүд [(12)](#ereduced-quart-khayyam) тэгшитгэлийн бодит шийдүүдийг өгнө гэдэг нь шууд харагдана. Нөгөө талаас, [(12)](#ereduced-quart-khayyam) тэгшитгэлийн ямар ч бодит шийд [(13)](#equart-khayyam) тэнцэтгэлийг хангах ба, оролцож байгаа бүх хувьсагчид болон коэффициентүүд нь бодит тул, хэрэв [(13)](#equart-khayyam) тэнцэтгэлийн баруун гар тал сөрөг тоо гарвал (ө.х. \({R}\) нь хуурмаг бол) анхны тэгшитгэл бодит шийдгүй болохын дохио гэсэн үг. Дээрх аргын нэг онцлог нь, \({y=x^2}\) гэсэн ганцхан параболыг л байгуулчихвал, үүнийгээ ашиглан ямар ч куб болон 4 зэргийн тэгшитгэлийг гортиг шугамаар бодож чаддаг болох юм.
