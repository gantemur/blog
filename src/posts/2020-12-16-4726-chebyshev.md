[Эрэмбэлэлтийн тэнцэл бишээс](/blog/2020/12/14/rearrangement/) арай дээгүүр, элементар чанараараа Кошитой ойролцоо зиндааны нэг чухал тэнцэл бишийг [1882 онд](https://archive.org/details/117744684_002) Оросын агуу математикч  [Пафнутий Львович Чебышёв](https://ru.wikipedia.org/wiki/Чебышёв,_Пафнутий_Львович)   баталсан нь одоо *Чебышёвын тэнцэл биш* (эсвэл Чебышёвын нийлбэрийн тэнцэл биш) гэсэн нэрээр алдаршжээ.

Энэ тэнцэл бишийн \({n=2}\) хувилбарыг гаргахын тулд эрэмбэлэлтийн тэнцэл биштэй ажиллаж байсантай төстэйгөөр, \({a_1\leq a_2}\) ба \({b_1\leq b_2}\) харьцааг хангах 4 тоо аваад, 

\(\displaystyle  (a_2-a_1)(b_2-b_1)\geq0\)

 тэнцэл бишийн зүүн гар талыг нь задалбал 

\(\displaystyle  a_2b_2+a_1b_1 \geq a_1b_2 + a_2b_1\)

 гарна. Одоо хоёр тал дээр нь \({a_2b_2+a_1b_1}\) илэрхийллийг нэмснээр 

\(\displaystyle  2(a_2b_2+a_1b_1) \geq (a_1+a_2)(b_1+b_2)\)

 буюу 

\(\displaystyle  \frac{a_1+a_2}2\cdot\frac{b_1+b_2}2 \leq \frac{a_1b_1+a_2b_2}2\)

 болно. Тэнцэлдээ хүрэх нөхцөл нь мэдээж \({a_1=a_2}\) эсвэл \({b_1=b_2}\).

Чебышёвын тэнцэл бишийг [Эмиль Пикар](https://en.wikipedia.org/wiki/Emile_Picard) яаж тайлбарласныг сонирхуулъя. Дээрх тэнцэл бишийг 

\(\displaystyle  \frac{x_1+x_2}2\leq \frac{x_1m_1+x_2m_2}{m_1+m_2}\)

 хэлбэрт бичээд, \({0
 [![](/blog/assets/wp-media/2020/12/centerofmass-1.png)](/blog/assets/wp-media/2020/12/centerofmass-1.png) 

Одоо Чебышёвын тэнцэл бишийг дурын \({n}\)-ийн хувьд баталъя.

**Теорем 1.**  Ямар ч \({a_1\leq a_2\leq \ldots\leq a_n}\) ба \({b_1\leq b_2\leq \ldots\leq b_n}\) хоёр дарааллын хувьд [

\(\displaystyle  \frac{a_1+\ldots+a_n}n\cdot\frac{b_1+\ldots+b_n}n\leq \frac{a_1b_1+\ldots+a_nb_n}n \ \ \ \ \ (1)\)

] тэнцэл биш биелэх ба, тэнцэлдээ хүрэх нөхцөл нь \({a_1=\ldots=a_n}\) эсвэл \({b_1=\ldots=b_n}\). 

Энэ теоремд \({b_1,\ldots,b_n}\) дарааллын оронд \({-b_1,\ldots,-b_n}\) дарааллыг авч үзсэнээр дараах мөрдлөгөө гарна.

**Мөрдлөгөө 2.**  Ямар ч \({a_1\leq a_2\leq \ldots\leq a_n}\) ба \({b_1\geq b_2\geq \ldots\geq b_n}\) хоёр дарааллын хувьд 

\(\displaystyle  \frac{a_1+\ldots+a_n}n\cdot\frac{b_1+\ldots+b_n}n \geq \frac{a_1b_1+\ldots+a_nb_n}n \ \ \ \ \ (2)\)

 тэнцэл биш биелэх ба, тэнцэлдээ хүрэх нөхцөл нь \({a_1=\ldots=a_n}\) эсвэл \({b_1=\ldots=b_n}\). 

 

##  Баталгаа 1 

 Ямар ч \({i,k}\) индексийн хувьд 

\(\displaystyle  (a_i-a_k)(b_i-b_k)\geq0\)

 гэж бичээд, хаалтыг нь нээж үржүүлбэл 

\(\displaystyle  a_ib_k+a_kb_i \leq a_ib_i+a_kb_k\)

 болно. Одоо \({i,k}\) индексүүдээр давхар нийлбэр авъя: 

\(\displaystyle  \sum_{i,k} (a_ib_k+a_kb_i) \leq \sum_{i,k} (a_ib_i+a_kb_k) .\)

 Үүний зүүн гар тал нь 

\(\displaystyle  \sum_{i,k} (a_ib_k+a_kb_i) = 2\sum_{i,k} a_ib_k = 2(a_1+\ldots+a_n) (b_1+\ldots+b_n)\)

 бол, баруун гар тал нь 

\(\displaystyle  \sum_{i,k} (a_ib_i+a_kb_k) = \sum_{i,k} a_ib_i + \sum_{i,k}a_kb_k =2n(a_1b_1+\ldots+a_nb_n) .\)

 Тэнцэл биш батлагдлаа.

Дээрх баталгаанаас, [(1)](#echebyshev) тэнцэл биш тэнцэлдээ хүрсэн нөхцөлд \({i,k}\) индекс бүрийн хувьд 

\(\displaystyle  (a_i-a_k)(b_i-b_k)=0\)

 байх нь тодорхой. Хэрэв \({a_1}\)-тэй тэнцүү биш бүх \({a_i}\)-ийн индексүүдийн олонлогийг \({I}\) гэвэл \({i\in I}\) болгоны хувьд \({b_i=b_1}\). Одоо \({j\in I}\) гэсэн нэг индекс бэхлээд, \({a_j}\)-тэй тэнцүү биш бүх \({a_k}\)-ийн индексүүдийн олонлогийг \({J}\) гэвэл, \({k\in J}\) болгоны хувьд \({b_k=b_j=b_1}\). Тодорхойлолт ёсоор \({J}\) нь \({I}\)-ийн гүйцээлтийг агуулна: \({I\cup J=\{1,\ldots,n\}}\). Иймд \({a_1=\ldots=a_n}\) биш бол \({b_1=\ldots=b_n}\) болж, теоремын баталгаа дууслаа.

##  Баталгаа 2 

Эрэмбэлэлтийн тэнцэл бишээс 

\(\displaystyle  \begin{array}{rcl}  a_1b_1 + a_2b_2\cdots + a_nb_n &\geq& a_1b_1 + a_2b_2\cdots + a_nb_n \\ a_1b_1 + a_2b_2\cdots + a_nb_n &\geq& a_1b_2 + a_2b_3 + \cdots + a_nb_1 \\ a_1b_1 + a_2b_2\cdots + a_nb_n &\geq& a_1b_3 + a_2b_3 + \cdots + a_n b_1 \\ \ldots&\ldots&\ldots\\ a_1b_1 + a_2b_2\cdots + a_nb_n &\geq& a_1b_n + a_2b_1 + \cdots a_nb_{n - 1} \end{array}\)

 гэж гарах ба, эдгээр тэнэл бишүүдийг бүгдийг нь хооронд нь нэмснээр 

\(\displaystyle  n(a_1b_1 + a_2b_2\cdots + a_nb_n)\geq (a_1 + a_2 + \cdots a_n)(b_1 + b_2 + \cdots b_n)\)

 болж, Чебышёвын тэнцэл биш батлагдана.

##  Жинтэй хувилбар 

Кошийн тэнцэл биштэй төстэйгөөр, Чебышёвын тэнцэл бишид жин оруулж бас болно.

**Теорем 3.**  Бодит тоон \({a_1\leq a_2\leq \ldots\leq a_n}\) ба \({b_1\leq b_2\leq \ldots\leq b_n}\) гэсэн хоёр дараалал, мөн \({p_1+\ldots+p_n=1}\) байх эерэг \({p_1,\ldots,p_n}\) тоонуудын хувьд 

\(\displaystyle  (p_1a_1+\ldots+p_na_n)(p_1b_1+\ldots+p_nb_n)\leq p_1a_1b_1+\ldots+p_na_nb_n \ \ \ \ \ (3)\)

 тэнцэл биш биелэх ба, тэнцэлдээ хүрэх нөхцөл нь \({a_1=\ldots=a_n}\) эсвэл \({b_1=\ldots=b_n}\). 

Теоремыг батлахын тулд ямар ч \({i,k}\) индексийн хувьд 

\(\displaystyle  (a_i-a_k)(b_i-b_k)\geq0\)

 гэж бичээд, хаалтыг нь нээж үржүүлбэл 

\(\displaystyle  a_ib_k+a_kb_i \leq a_ib_i+a_kb_k\)

 болно. Үүнийг \({p_ip_k}\)-ээр үржүүлээд, \({i,k}\) индексүүдээр давхар нийлбэр авъя: 

\(\displaystyle  \sum_{i,k} p_ip_k(a_ib_k+a_kb_i) \leq \sum_{i,k} p_ip_k(a_ib_i+a_kb_k) .\)

 Одоо зүүн гар тал нь 

\(\displaystyle  \sum_{i,k} p_ip_k(a_ib_k+a_kb_i) = 2\sum_{i,k} p_ip_ka_ib_k = 2\sum_{i} p_ia_i \sum_{k} p_kb_k ,\)

 баруун гар тал нь 

\(\displaystyle  \sum_{i,k} p_ip_k(a_ib_i+a_kb_k) = \sum_{i,k} p_kp_ia_ib_i + \sum_{i,k}p_ip_ka_kb_k =2\sum_{i} p_ia_ib_i .\)

 болохыг ажиглавал, тэнцэл биш батлагдана. Тэнцэлдээ хүрэх нөхцлийн хувьд Баталгаа 1-тэй яг адилхан.

##  Жишээнүүд 

**Жишээ 4.**  Дурын эерэг \({x,y,z}\) тоонуудын хувьд 

\(\displaystyle  \frac{x^8}{yz} + \frac{y^8}{zx} + \frac{z^8}{xy} \geq x^6+y^6+z^6\)

 гэж харуул. 

*Бодолт.*  Тэнцэл биш маань \({x}\), \({y}\), \({z}\) хувьсагчдын хувьд тэгшхэмтэй тул \({x\leq y\leq z}\) гэж үзэхэд алдах юм байхгүй. Ингээд Чебышёвын болон Кошийн тэнцэл бишүүдийг угсруулан хэрэглэвэл 

\(\displaystyle  x^9+y^9+z^9 \geq \frac{x^3+y^3+z^3}3\cdot(x^6+y^6+z^6) \geq xyz\cdot(x^6+y^6+z^6)\)

 болох ба хоёр талыг нь \({xyz}\) үржвэрт хувааснаар бидний зорилго биелнэ. \(\Box\)

**Жишээ 5.**  Нийлбэр нь \({n}\)-тэй тэнцүү, эерэг \({x_1,\ldots,x_n}\) тоонуудын хувьд 

\(\displaystyle  x_1^n+\ldots+x_n^n \leq x_1^{n+1}\ldots+x_n^{n+1}\)

 гэж харуул. 

*Бодолт.*  Чебышёвын тэнцэл бишийг \({(x_1,\ldots,x_n)}\) ба \({(x_1^n,\ldots,x_n^n)}\) дарааллуудын эрэмбэлэгдсэн хувилбар дээр хэрэглэснээр 

\(\displaystyle  \frac{x_1+\ldots+x_n}n \cdot \frac{x_1^n+\ldots+x_n^n}n \leq \frac{x_1^{n+1}\ldots+x_n^{n+1}} n\)

 болж, бидний батлах гэсэн тэнцэл биш гарч ирнэ. \(\Box\)

**Жишээ 6.**  Нийлбэр нь \({3}\)-тай тэнцүү, эерэг \({a,b,c}\) тоонуудын хувьд 

\(\displaystyle  \frac1{a+b+c^2} + \frac1{b+c +a^2} + \frac1{c+a+b^2} \leq 1\)

 гэж харуул. 

*Бодолт.*  Бодлогын нөхцлөөс жишээ нь \({a+b=3-c}\) тул \({a+b+c^2=c^2-c+3}\). Тэгэхээр батлах ёстой тэнцэл бишийг 

\(\displaystyle  \frac1{c^2-c+3} + \frac1{a^2-a+3} + \frac1{b^2-b+3} \leq 1\)

 гэж бичиж болно. Одоо баруун гар талын 1-ийг зүүн гар талын 3 гишүүнд жигд хуваарилан хасвал 

\(\displaystyle  \frac{c-c^2}{c^2-c+3} + \frac{a-a^2}{a^2-a+3} + \frac{b-b^2}{b^2-b+3} \leq 0\)

 буюу 

\(\displaystyle  \frac{a-1}{a-1+\frac3a} + \frac{b-1}{b-1+\frac3b} + \frac{c-1}{c-1+\frac3c} \geq 0\)

 болж хувирна. Тэнцэл биш маань \({a,b,c}\)-ийн хувьд тэгш хэмтэй тул мэдээж \({a\leq b\leq c}\) гэж үзэхэд алдах юм байхгүй. Эндээс 

\(\displaystyle  a-1+\frac3a\geq b-1+\frac3b\geq c-1+\frac3c\)

 гэж гарна: Тухайлбал \({a-1+\frac3a\geq b-1+\frac3b}\) нь \({(a-b)(ab-3)\geq0}\) нөхцөлтэй адилхан. Сүүлийн нөхцлийг Кошийн тэнцэл бишээр 

\(\displaystyle  ab\leq\Big(\frac{a+b}2\Big)^2<\big(\frac32\big)^2<3\)

 гэж шалгаж болно. Ингээд Чебышёвын тэнцэл бишээс 

\(\displaystyle  \frac{a-1}{a-1+\frac3a} + \frac{b-1}{b-1+\frac3b} + \frac{c-1}{c-1+\frac3c} \geq (a-1+b-1+c-1)\cdots=0\)

 гэж гарах тул бодлого бодогдлоо. \(\Box\)
