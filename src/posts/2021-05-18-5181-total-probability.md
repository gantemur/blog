Үзэгдлийн магадлалыг олоход өргөн ашиглагддаг нэг арга нь нөхцөлт магадлалуудаар нь дамжуулж тооцох явдал юм. Тухайлбал, дурын \\({A}\\), \\({B}\\) үзэгдлийн хувьд

\\(\displaystyle  A = A\cap\Omega = A\cap(B\cup\bar B) = AB \cup A\bar B\\)

 ба \\({(AB) \cap (A\bar B)=\emptyset}\\) гэдгээс *бүтэн магадлалын томъёо* гэгдэх томъёо гарна:

\\(\displaystyle  P(A) = P(AB)+P(A\bar B) = P(A|B)P(B) + P(A|\bar B)P(\bar B) .\\)

 Тэгэхээр \\({P(A|B)}\\) ба \\({P(A|\bar B)}\\) гэсэн нөхцөлт магадлалуудыг \\({P(B)}\\) магадлалын хамтаар мэдэж байвал \\({P(A)}\\) магадлал шууд олдоно.

Бүтэн магадлалын томьёонд орсон \\({B,\bar B}\\) гэсэн хоёр үзэгдийг хос хосоороо нийцгүй,

\\(\displaystyle  B_1\cup B_2\cup\ldots\cup B_n=\Omega\\)

 байдаг \\({B_1,B_2,\ldots,B_n}\\) гэсэн дурын \\({n}\\) ширхэг үзэгдлээр сольж болно. Ийм үзэгдлүүдийн олонлогийг *гүйцэд систем* гэж ярьдаг. Тэгвэл

\\(\displaystyle  A = A\cap(B_1\cup B_2\cup\ldots\cup B_n) = AB_1\cup AB_2\cup\ldots\cup AB_n\\)

 ба \\({AB_1,\ldots,AB_n}\\) үзэгдлүүд хос хосоороо нийцгүй тул магадлалыг нэмэх дүрмээр

\\(\displaystyle  P(A) = P(AB_1)+\ldots+P(AB_n) = P(A|B_1) P(B_1) +\ldots+ P(A|B_n)P(B_n) .\\)

 Үүнийг нийлбэрийн тэмдэг ашиглан товчилж бичвэл

\\(\displaystyle  P(A) = \sum_{k=1}^n P(A|B_k) P(B_k)\\)

 болно.

**Жишээ 1.**  Эхний хайрцганд 4 хар, 6 цагаан бөмбөг, хоёрдахь хайрцганд 5 хар, 4 цагаан бөмбөг байсан ба эхний хайрцагнаас санамсаргүйгээр нэг бөмбөг авснаа өнгийг нь харалгүйгээр хоёрдахь хайрцагт хийсний дараа хоёрдахь хайрцгаас нэг бөмбөг санамсаргүйгээр сонгов. Энэ бөмбөг цагаан байх магадлалыг ол.

*Бодолт.*  Эхний хайрцагнаас хар бөмбөг гарч ирэх үзэгдлийг \\({B}\\), эцсийн бөмбөг цагаан байх магадлалыг \\({A}\\) гэе. Тэгвэл эхний хайрцагнаас цагаан бөмбөг гарч ирэх үзэгдэл \\({\bar B}\\) болох ба

\\(\displaystyle  P(B) = \frac4{10}, \qquad P(\bar B) = 1-P(B) = \frac6{10},\\)

 байх нь ойлгомжтой. Түүнчлэн

\\(\displaystyle  P(A|B) = \frac{4}{9+1} = \frac{4}{10}, \qquad P(A|\bar B) = \frac{4+1}{9+1} = \frac{5}{10},\\)

 тул бүтэн магадлалын томъёо ёсоор

\\(\displaystyle  P(A) = P(A|B)P(B) + P(A|\bar B)P(\bar B) = \frac{4}{10}\cdot\frac4{10} + \frac{5}{10}\cdot\frac6{10} = \frac{46}{100}\\)

 байна. \\(\Box\\)

**Жишээ 2.**  Та 3 ширхэг буутай ба эдгээрээр буудвал таны бай онох магадлал харгалзан 0.5, 0.62, 0.8. Хэрэв та нэг бууг нь санамсаргүй сонгоод бай руу буудсан бол онох магадлал хэд вэ?

*Бодолт.*  Гурван буу сонгогдох үзэгдлүүдийг харгалзан \\({B_1}\\), \\({B_2}\\), \\({B_3}\\) гэвэл эдгээр нь гүйцэд систем болох ба

\\(\displaystyle  P(B_1) = P(B_2) = P(B_3) = \frac13.\\)

 Буудагч байг онох үзэгдлийг \\({A}\\) гэвэл бүтэн магадлалын томъёогоор

\\(\displaystyle  P(A) = P(A|B_1) P(B_1) + P(A|B_2) P(B_2) + P(A|B_3) P(B_3) \\ {}\qquad\qquad= \frac{0.5+0.62+0.8}3 = 0.64\\)

  гарна. \\(\Box\\)

**Жишээ 3 (Тоглоомчны дампуурал).**  Мөрийтэй тоглоомын газар ирсэн нэг хүн зоос шидээд сүлдээрээ буувал 1 төгрөг хождог, тоогоороо буувал 1 төгрөг алддаг тоглоом тоглож эхлэв. Зоосны сүлдээрээ буух магадлал \\({0<\lambda<1}\\). Тэр хүн анх \\({k}\\) төгрөгтэй байсан ба нэг бол бүх мөнгөө алдаж дуусах, эсвэл нийт \\({m}\\) төгрөгтэй болох хүртлээ тоглоно гэсэн төлөвлөгөөтэй байв. Түүний бүх мөнгөө алдах магадлалыг ол.

*Бодолт.*  Тоглоомчин эцэстээ бүх мөнгөө алдах үзэгдлийг \\({A}\\), эхний алхамдаа хожих үзэгдлийг \\({B}\\) гэвэл \\({P(B)=\lambda}\\) тул

\\(\displaystyle  P(A) = P(A|B)P(B) + P(A|\bar B)P(\bar B) = \lambda P(A|B) + (1-\lambda)P(A|\bar B) .\\)

 Цаашилбал, анх \\({k}\\) төгрөгтэй эхлээд бүх мөнгөө алдах магадлалыг \\({f(k)=P(A)}\\) гэвэл \\({P(A|B)=f(k+1)}\\) ба \\({P(A|B)=f(k-1)}\\) байна. Үүнийг дээрх томъёонд орлуулбал

\\(\displaystyle  f(k) = \lambda f(k+1) + (1-\lambda)f(k-1)\\)

 тул \\({f}\\) нь шугаман функц байх ёстой:

\\(\displaystyle  f(x) = ax+b .\\)

 Анх 0 төгрөгтэй эхэбэл шууд дампуурна, харин анх \\({m}\\) төгрөгтэй эхэлбэл дампуурах магадлал 0:

\\(\displaystyle  f(0) = 1, \qquad f(m) = 0\\)

 Эндээс \\({a}\\), \\({b}\\) тогтмолуудаа олбол

\\(\displaystyle  f(x) = 1 - \frac{x}m\\)

 буюу манай бодлогын хариу

\\(\displaystyle  P(A) = 1 - \frac{k}m\\)

 гарна. \\(\Box\\)
