[Зэрэгт дунджуудын](/blog/2020/12/04/power-mean/) тэнцэл биш [Кошийн](/blog/2020/12/03/amgm/) тэнцэл бишийг яаж өргөтгөж байсантай төстэйгөөр [Коши-Буняковскийн](/blog/2020/12/07/cbs/) тэнцэл бишийг өргөтгөдөг тэнцэл бишийг [Гёльдерийн тэнцэл биш](https://en.wikipedia.org/wiki/Holder%27s_inequality) гэдэг. Энэ тэнцэл биш нь Кошийн болон Коши-Буняковскийн тэнцэл биштэй адилаар «галын шугаманд» арай явдаггүй боловч «2-р эгнээнд» бэлэн байдалтай байж байдаг тэнцэл бишүүдийн нэг юм.

Гёльдерийн тэнцэл бишийг анх Британийн математикч [Леонард Жеймс Рожерс](https://en.wikipedia.org/wiki/Leonard_James_Rogers) [1888 онд](https://archive.org/stream/messengermathem01unkngoog#page/n183/mode/1up), Германы математикч [Отто Гёльдер](https://en.wikipedia.org/wiki/Otto_Holder) [1889 онд](http://www.digizeitschriften.de/dms/img/?PID=GDZPPN00252421X) өөр өөр аргаар тус тус баталсан. Харин жинхэнэ үнэ цэнийг нь ойлгож математикт нэвтрүүлсэн хүн бол Унгарын агуу математикч [Фридьеш Рис](https://en.wikipedia.org/wiki/Frigyes_Riesz) юм.

**Теорем 1 (Гёльдер).**  Хэрэв \({p,q}\) нь \({\frac1p+\frac1p=1}\) нөхцлийг хангах эерэг рационал тоонууд бол дурын \({x_1,y_1,x_2,y_2,\ldots,x_n,y_n}\) тоонуудын хувьд [

\(\displaystyle  x_1y_1+\ldots+x_ny_n \leq \big(|x_1|^p+\ldots+|x_n|^p\big)^{\frac1p} \big(|y_1|^q+\ldots+|y_n|^q\big)^{\frac1q} \ \ \ \ \ (1)\)

] тэнцэл биш биелэх ба, тэнцэлдээ хүрсэн нөхцөлд [

\(\displaystyle  |x_1|^{p}=\lambda|y_1|^{q},\quad\ldots,\quad |x_n|^{p}=\lambda|y_n|^{q}, \ \ \ \ \ (2)\)

] байх \({\lambda}\) гэсэн тоо олдоно, эсвэл \({y_1=y_2=\ldots=y_n=0}\) байна. 

Дээрх теоремд \({p,q}\) тоонуудыг рационал байхыг шаарддаг нь бид баталгаандаа зөвхөн алгебрын аргыг хэрэглэнэ гэж зүтгэсний үр дагавар юм. Үнэндээ бол ганц хязгаар аваад энэ тэнцэл бишийг бодит \({p,q}\)-ийн хувьд өргөтгөж болно. 

##  Баталгаа 1 

[Коши-Буняковскийн](/blog/2020/12/07/cbs/) тэнцэл бишийг баталж байсан аргууд дотроос квадратлаг чанар дээр хэт түшиглэсэн аргууд энд ажиллахгүй. Харин [Кошийн](/blog/2020/12/03/amgm/) тэнцэл бишийг [Юнгийн](/blog/2020/12/03/amgm/) тэнцэл бишээр солиод Кошийн тэнцэл биш дээр тулгуурласан аргуудыг өргөтгөх гэж оролдвол ирээдүйтэй байж магад. Энэ аргыг анх Фридьеш Рис 1928 онд хэрэглэжээ.

Тэгэхээр [(1)](#eholder) тэнцэл бишийн зүүн гар талын гишүүн бүр дээр Юнгийн тэнцэл бишийг \({t>0}\) гэсэн чөлөөт параметртэйгээр хэрэглэвэл [

\(\displaystyle  x_iy_i\leq|x_iy_i|=\frac{|x_i|}t\cdot|y_i|t\leq\frac{t^{-p}|x_i|^p}{p}+\frac{t^q|y_i|^q}q \ \ \ \ \ (3)\)

] болно. Тэнцэлдээ хүрэх нөхцөл нь \({t^{-p}|x_i|^p=t^q|y_i|^q}\) гэдгийг бас санацгаая.

Одоо эдгээрийг нийлбэрчилбэл [

\(\displaystyle  x_1y_1+\ldots+x_ny_n \leq \frac{t^{-p}\big(|x_1|^p+\ldots+|x_n|^p\big)}{p} +\frac{t^q\big( |y_1|^q+\ldots+|y_n|^q \big)}q \ \ \ \ \ (4)\)

] гарах ба Юнгийн тэнцэл бишийн тэнцэлдээ хүрэх нөхцлөөс санаа аван 

\(\displaystyle  t^{-p}\big(|x_1|^p+\ldots+|x_n|^p\big) ={t^q}\big( |y_1|^q+\ldots+|y_n|^q \big)\)

 байхаар \({t}\) параметрийн утгыг сонгосноор 

\(\displaystyle  t^{p+q} = \frac{|x_1|^p+\ldots+|x_n|^p}{|y_1|^q+\ldots+|y_n|^q}\)

 буюу 

\(\displaystyle  t^q = \Big( \frac{|x_1|^p+\ldots+|x_n|^p}{|y_1|^q+\ldots+|y_n|^q} \Big)^{\frac1{p}}, \qquad t^{-p} = \Big( \frac{|y_1|^q+\ldots+|y_n|^q}{|x_1|^p+\ldots+|x_n|^p} \Big)^{\frac1{q}}\)

 гэсэн үр дүнд хүрнэ. Энд 

\(\displaystyle  1 = \frac1p+\frac1q=\frac{p+q}{pq} \qquad\Longrightarrow\qquad p+q=pq\)

 гэдгийг ашигласан. Ингээд \({t^{-p}}\) ба \({t^q}\) илэрхийллүүдээ [(4)](#eholder-pf-1) томъёоны баруун гар талд орлуулбал бидний батлах гэсэн [(1)](#eholder) тэнцэл биш гарч ирнэ. Энэ тэнцэл биш тэнцэлдээ хүрсэн үед [(3)](#eholder-pf-0-young) тэнцэл биш бүр бүгд тэнцэлдээ хүрэх тул [(2)](#eholder-equality) нөхцөл мөрдөнө.

Эцэст нь, \({x_1=\ldots=x_n=0}\) эсвэл \({y_1=\ldots=y_n=0}\) үед [(1)](#eholder) тэнцэл биш тривиал байдлаар биелэх тул эдгээр тохиолдлыг бид баталгаандаа авч үзэхгүй хассан нь асуудал үүсгэхгүй гэдгийг тэмдэглэе.

##  Баталгаа 2 

Дээрх баталгааг жаахан өөрчлөөд [жинтэй Кошийн тэнцэл биш](/blog/2020/12/03/amgm/) дээр суурилсан баталгааг бас байгуулж болдог. Үүний тулд юуны түрүүнд, Гёльдерийн [(1)](#eholder) тэнцэл биш нь рациональ тоо \({0<\lambda<1}\) ба сөрөг биш \({a_i}\), \({b_i}\) тоонуудын хувьд биелэх [

\(\displaystyle  a_1^{\lambda}b_1^{1-\lambda}+\ldots+a_n^{\lambda}b_n^{1-\lambda} \leq \big(a_1+\ldots+a_n\big)^{\lambda} \big(b_1+\ldots+b_n\big)^{1-\lambda} \ \ \ \ \ (5)\)

] тэнцэл биштэй эквивалент гэдгийг ажиглая.

Хэрэв \({\sum_ia_i=0}\) эсвэл \({\sum_ib_i=0}\) бол [(5)](#eholder-alt) тэнцэл биш биелэх нь тодорхой. Иймд эдгээр нийлбэрийг тэг биш гэж үзээд, 

\(\displaystyle  \alpha_i = \frac{a_i}{\sum_ia_i}, \qquad \beta_i = \frac{b_i}{\sum_ib_i},\)

 гэсэн тоонууд дээр жинтэй Кошийн тэнцэл бишийг хэрэглэвэл 

\(\displaystyle  \sum_i \alpha_i^\lambda \beta_i^{1-\lambda} \leq \lambda\sum_i\alpha_i + (1-\lambda)\sum_i\beta_i =1\)

 гарах ба эндээс шууд [(5)](#eholder-alt) тэнцэл биш мөрдөнө.

Дээрх аргыг өргөтгөөд \({\lambda+\mu+\nu=1}\) байх гурван эерэг рациональ тоо ба сөрөг биш \({a_i}\), \({b_i}\), \({c_i}\) дарааллуудын хувьд [

\(\displaystyle  \sum_i a_i^{\lambda}b_i^{\mu}c_i^{\nu} \leq \big(\sum_ia_i\big)^{\lambda} \big(\sum_ib_i\big)^{\mu} \big(\sum_ic_i\big)^{\nu} \ \ \ \ \ (6)\)

] буюу \({\frac1p+\frac1q+\frac1r=1}\) байх гурван эерэг рациональ тооны хувьд 

\(\displaystyle  \sum_i a_ib_ic_i \leq \big(\sum_ia_i^p\big)^{\frac1p} \big(\sum_ib_i^q\big)^{\frac1q} \big(\sum_ic_i^r\big)^{\frac1r} \ \ \ \ \ (7)\)

 гэсэн Гёльдерийн *өргөтгөсөн* тэнцэл бишийг гарган авахад амархан. Мэдээж үүнийг цааш нь хэдэн ч ширхэг дарааллын хувьд өргөтгөж болно.

##  Баталгаа 3 

Гёльдерийн [(1)](#eholder) тэнцэл бишийн \({p=q=2}\) байх тухайн тохиолдол нь мэдээж [Коши-Буняковский](/blog/2020/12/07/cbs/). Сонирхолтой нь, Гёльдерийг энэ тухайн тохиолдлынх нь мөрдлөгөө мэтээр баталж бас болдог. Коши-Буняковскийг 2 удаа хэрэглэн  

\(\displaystyle  \sum_i a_ib_ic_id_i \leq \Big( \sum_i a_i^2b_i^2 \Big)^{\frac12} \Big( \sum_i c_i^2d_i^2 \Big)^{\frac12} \\ {}\qquad\qquad\leq \Big( \sum_i a_i^4 \Big)^{\frac14} \Big( \sum_i b_i^4 \Big)^{\frac14} \Big( \sum_i c_i^4 \Big)^{\frac14} \Big( \sum_i d_i^4 \Big)^{\frac14}\)

  буюу, \({a_i,b_i,c_i,d_i}\) бүгд сөрөг биш үед 

\(\displaystyle  \sum_i a_i^{\frac14}b_i^{\frac14}c_i^{\frac14}d_i^{\frac14} \leq \Big( \sum_i a_i \Big)^{\frac14} \Big( \sum_i b_i \Big)^{\frac14} \Big( \sum_i c_i \Big)^{\frac14} \Big( \sum_i d_i \Big)^{\frac14}\)

 гэж гаргаж болно. Энд \({a_i=b_i=c_i}\) гэж авбал 

\(\displaystyle  \sum_i a_i^{\frac34}d_i^{\frac14} \leq \Big( \sum_i a_i \Big)^{\frac34} \Big( \sum_i d_i \Big)^{\frac14} .\)

 Өөрөөр хэлбэл Гёльдерийн [(5)](#eholder-alt) тэнцэл биш \({\lambda=\frac{k}4}\) (буюу \({p=\frac4k}\)) хэлбэрийн илтгэгчдийн хувьд хүчинтэй. Цаашилбал, дээрхтэй төстэйгөөр Коши-Буняковскийг давтан хэрэглэснээр Гёльдерийн [(5)](#eholder-alt) тэнцэл биш \({\lambda=\frac{k}{2^m}}\) хэлбэрийн илтгэгчдийн хувьд хүчинтэй болохыг баталж болно.

Одоо дурын \({\lambda=\frac{k}{N}}\) хэлбэрийн илтгэгчдийн хувьд дээрх баталгааг өргөтгөхийн тулд, \({N}\) ширхэг дарааллын хувьд биелэх [

\(\displaystyle  \sum_i a_i^{\frac1N}b_i^{\frac1N}\cdots \leq \Big( \sum_i a_i \Big)^{\frac1N} \Big( \sum_i b_i \Big)^{\frac1N}\cdots \ \ \ \ \ (8)\)

] тэнцэл бишээс \({N-1}\) ширхэг дарааллын хувьд биелэх хувилбарыг нь гарган авахад хангалттай. Бичиглэлийг хялбарчлах үүднээс үүний үндсэн санааг \({N=6}\) үед тайлбарлая. Эхлээд \({a_i}\), \({b_i}\), \({c_i}\), \({d_i}\), \({e_i}\) гэсэн 5 ширхэг дарааллын хувьд 

\(\displaystyle  \sum_i a_i^{\frac15}b_i^{\frac15}c_i^{\frac15}d_i^{\frac15}e_i^{\frac15} = \sum_i a_i^{\frac16}b_i^{\frac16}c_i^{\frac16}d_i^{\frac16}e_i^{\frac16} \cdot \big(a_i^{\frac15}b_i^{\frac15}c_i^{\frac15}d_i^{\frac15}e_i^{\frac15}\big)^{\frac16}\)

 гэж бичээд, [(8)](#eholder-cbs-N) тэнцэл бишийн \({N=6}\) хувилбарыг хэрэглэн  

\(\displaystyle  \sum_i a_i^{\frac15}b_i^{\frac15}c_i^{\frac15}d_i^{\frac15}e_i^{\frac15} = \sum_i a_i^{\frac16}b_i^{\frac16}c_i^{\frac16}d_i^{\frac16}e_i^{\frac16} \cdot \big(a_i^{\frac15}b_i^{\frac15}c_i^{\frac15}d_i^{\frac15}e_i^{\frac15}\big)^{\frac16}\\ {}\qquad\leq \Big( \sum_i a_i \Big)^{\frac16} \Big( \sum_i b_i \Big)^{\frac16} \Big( \sum_i c_i \Big)^{\frac16} \Big( \sum_i d_i \Big)^{\frac16} \Big( \sum_i e_i \Big)^{\frac16} \Big( \sum_i a_i^{\frac15}b_i^{\frac15}c_i^{\frac15}d_i^{\frac15}e_i^{\frac15} \Big)^{\frac16}\)

  гэдгийг харна. Энэ мөчид \({\sum_i a_ib_ic_id_ie_i\neq0}\) гэж үзээд, хоёр талыг нь хамгийн сүүлийн үржигдэхүүнд хувааснаар 

\(\displaystyle  \Big( \sum_i a_i^{\frac15}b_i^{\frac15}c_i^{\frac15}d_i^{\frac15}e_i^{\frac15} \Big)^{\frac56} \leq \Big( \sum_i a_i \Big)^{\frac16} \Big( \sum_i b_i \Big)^{\frac16} \Big( \sum_i c_i \Big)^{\frac16} \Big( \sum_i d_i \Big)^{\frac16} \Big( \sum_i e_i \Big)^{\frac16}\)

 буюу 

\(\displaystyle  \sum_i a_i^{\frac15}b_i^{\frac15}c_i^{\frac15}d_i^{\frac15}e_i^{\frac15} \leq \Big( \sum_i a_i \Big)^{\frac15} \Big( \sum_i b_i \Big)^{\frac15} \Big( \sum_i c_i \Big)^{\frac15} \Big( \sum_i d_i \Big)^{\frac15} \Big( \sum_i e_i \Big)^{\frac15}\)

 болж, \({6\rightarrow5}\) гэсэн шилжилт батлагдана.

Эцэст нь дүгнэхэд, Коши-Буняковскийг давтан хэрэглэснээр [(8)](#eholder-cbs-N) тэнцэл бишийн \({N=2^m}\) илтгэгчтэй хувилбарууд шууд гарах ба, үүнээсээ \({2^m\rightarrow2^m-1\rightarrow2^m-2\ldots}\) гэх мэтчилэн хөөвөл [(8)](#eholder-cbs-N) тэнцэл биш дурын \({N}\) тооны хувьд хүчинтэй нь батлагдана. Энэ нь Гёльдерийн [(5)](#eholder-alt) тэнцэл бишийг дурын \({\lambda=\frac{k}N}\) илтгэгчийн хувьд төрүүлээд зогсохгүй, өргөтгөсөн [(6)](#eholder-gen-alt) маягийн хувилбаруудыг нь ч батлахад хангалттай юм.

##  Хосмог чанар 

[Коши-Буняковскийн тэнцэл бишийг](/blog/2020/12/07/cbs/) 

\(\displaystyle  \langle x,y\rangle \leq\|x\|\|y\|\)

 хэлбэртэй бичиж болдог. Үүнээс санаа аваад 

\(\displaystyle  \langle x,y\rangle = x_1y_1+\ldots+x_ny_n, \qquad \|x\|_p=\Big(|x_1|^p+\ldots+|x_n|^p\Big)^{\frac1p},\)

 гэсэн тэмдэглэгээнүүдийг оруулан Гёльдерийн тэнцэл бишийг 

\(\displaystyle  \langle x,y\rangle \leq\|x\|_p\|y\|_q, \qquad \frac1p+\frac1q=1,\)

 маягтай бичиж болно. Коши-Буняковскийн тохиолдолд (ө.х. \({p=q=2}\) үед) 

\(\displaystyle  \|x\|_2 = \sqrt{\langle x,x\rangle}\)

 байдаг нь онцгой. Энэ чанарыг ерөнхий \({p}\)-тэй тохиолдолд зарим талаар орлож чадах нэг чанар байдгийг Германы алдарт математикч [Эдмунд Ландау](https://en.wikipedia.org/wiki/Edmund_Landau) 1907 онд нээжээ.

**Теорем 2.**  Хэрэв \({p,q}\) нь \({\frac1p+\frac1p=1}\) нөхцлийг хангах эерэг тоонууд, \({M}\) нь сөрөг биш тоо, \({x\in{\mathbb R}^n}\) нь өгөгдсөн вектор бөгөөд дурын \({y\in{\mathbb R}^n}\) векторын хувьд [

\(\displaystyle  \langle x,y\rangle \leq M\|y\|_q \ \ \ \ \ (9)\)

] тэнцэл биш биелдэг бол 

\(\displaystyle  \|x\|_p\leq M\)

 байна. 

Үүнийг батлахын тулд 

\(\displaystyle  y_i=\mathrm{sign}(x_i)|x_i|^{1/(q-1)}\)

 гэсэн сонголт хийвэл 

\(\displaystyle  \langle x,y\rangle = \sum_i x_i\mathrm{sign}(x_i)|x_i|^{1/(q-1)} = \sum_i |x_i|^{q/(q-1)}\)

 ба 

\(\displaystyle  \|y\|_q^q = \sum_i |x_i|^{q/(q-1)} .\)

 

\(\displaystyle  \frac{q}{q-1}=\frac1{1-\frac1{q}}=\frac1p\)

 Одоо 

\(\displaystyle  \sum_i |x_i|^{1/p} \leq M \Big( \sum_i |x_i|^{1/p} \Big)^{\frac1q}\)

 гэдгийг тооцоод, [(9)](#eriesz-dual) нөхцлийг бичвэл 

\(\displaystyle  \Big( \sum_i |x_i|^{1/p} \Big)^{1-\frac1q} \leq M\)

 болж, теорем батлагдана. Сүүлийн алхамд \({\|x\|_p\neq0}\) гэж үзсэн.

 

##  Жишээнүүд 

**Жишээ 3.**  Эерэг \({a,b,c}\) тоонууд \({a+b+c=1}\) нөхцлийг хангадаг бол 

\(\displaystyle  a\sqrt[3]{1+b-c} + b\sqrt[3]{1+c-a} + c\sqrt[3]{1+a-b} \leq 1\)

 гэж харуул. 

*Бодолт.*  Тэнцэл бишийн зүүн гар талыг 

\(\displaystyle  \sum_{\mathrm{cyc}}a\sqrt[3]{1+b-c} \leq \sum_{\mathrm{cyc}}a^{2/3}[a(1+b-c)]^{1/3}\)

 гэж бичээд, Гёльдерийн тэнцэл бишийг \({p=\frac32}\) ба \({q=3}\) илтгэгчидтэй хэрэглэвэл 

\(\displaystyle  \sum_{\mathrm{cyc}}a\sqrt[3]{1+b-c} \leq \Big( \sum_{\mathrm{cyc}} a\Big)^{\frac23} \Big( \sum_{\mathrm{cyc}} a(1+b-c)\Big)^{\frac13}\)

 болно. Энд \({\sum_{\mathrm{cyc}}}\) тэмдэглэгээ нь \({a\rightarrow b\rightarrow c\rightarrow a}\) маягаар «циклдэж» нийлбэр авахыг илэрхийлж байгаа. Тухайлбал 

\(\displaystyle  \sum_{\mathrm{cyc}}a = a + b +c =1 ,\)

 ба 

\(\displaystyle  \sum_{\mathrm{cyc}}a(1+b-c) = a(1+b-c) + b(1+c-a) + c(1+a-b) = a + b +c =1 ,\)

 тул, бидний батлах гэсэн тэнцэл биш батлагдана. \(\Box\)

**Жишээ 4.**  Дурын эерэг \({a,b,c}\) тоонууд ба \({n\geq1}\) бүхэл тооны хувьд 

\(\displaystyle  \frac{a^{n}}{b^{n-1}} + \frac{b^{n}}{c^{n-1}} + \frac{c^{n}}{a^{n-1}} \leq \frac{a^{n+1}}{b^{n}} + \frac{b^{n+1}}{c^{n}} + \frac{c^{n+1}}{a^{n}}\)

 гэж харуул. 

*Бодолт.*  Гёльдерийн тэнцэл бишийг \({p=\frac{n}{n-1}}\) ба \({q=n}\) илтгэгчтэйгээр 

\(\displaystyle  \Big(\frac{a^{(n^2-1)/n}}{b^{n-1}},\frac{b^{(n^2-1)/n}}{c^{n-1}},\frac{c^{(n^2-1)/n}}{a^{n-1}}\Big), \qquad (a^{1/n},b^{1/n},c^{1/n})\)

 дарааллууд дээр хэрэглэвэл [

\(\displaystyle  \frac{a^{n}}{b^{n-1}} + \frac{b^{n}}{c^{n-1}} + \frac{c^{n}}{a^{n-1}} \leq \Big( \frac{a^{n+1}}{b^{n}} + \frac{b^{n+1}}{c^{n}} + \frac{c^{n+1}}{a^{n}} \Big)^{\frac{n-1}n} \big(a+b+c\big)^{\frac1n} \ \ \ \ \ (10)\)

] гарна. Одоо Гёльдерийн тэнцэл бишийг дахиад \({p=\frac{n}{n-1}}\) ба \({q=n}\) илтгэгчтэйгээр 

\(\displaystyle  (b^{(n-1)/n},c^{(n-1)/n},a^{(n-1)/n}) , \qquad \Big(\frac{a}{b^{(n-1)/n}},\frac{b}{c^{(n-1)/n}},\frac{c}{a^{(n-1)/n}}\Big)\)

 дарааллууд дээр хэрэглэвэл 

\(\displaystyle  a+b+c \leq \big(b+c+a\big)^{\frac{n-1}n} \Big( \frac{a^{n}}{b^{n-1}} + \frac{b^{n}}{c^{n-1}} + \frac{c^{n}}{a^{n-1}} \Big)^{\frac{1}n}\)

 буюу 

\(\displaystyle  \big(a+b+c\big)^{\frac{1}n} \leq \Big( \frac{a^{n}}{b^{n-1}} + \frac{b^{n}}{c^{n-1}} + \frac{c^{n}}{a^{n-1}} \Big)^{\frac{1}n}\)

 болно. Үүнийгээ [(10)](#eholder-eg-1) томъёонд орлуулснаар баталгаа дуусна. \(\Box\)

**Жишээ 5.**  Сөрөг биш \({a,b,c}\) тоонуудын хувьд 

\(\displaystyle  \frac{a+\sqrt{ab}+\sqrt[3]{abc}}3 \leq \sqrt[3]{a\cdot\frac{a+b}2\cdot\frac{a+b+c}3}\)

 болохыг харуул. 

*Бодолт.*  Юуны өмнө \({(\sqrt[3]{a^2},\sqrt[3]{ab},\sqrt[3]{ac}\,)}\) ба \({(\sqrt[3]{a},\sqrt[6]{ab},\sqrt[3]{b}\,)}\) векторууд дээр Гёльдерийн тэнцэл бишийг \({p=\frac32}\) (ба \({q=3}\)) илтгэгчтэйгээр хэрэглэвэл 

\(\displaystyle  a+\sqrt{ab}+\sqrt[3]{abc} \leq \big( a+\sqrt{ab}+\sqrt{ac}\big)^{\frac23} \big( a+\sqrt{ab}+b \big)^{\frac13}\)

 гарна. Одоо баруун гар талын эхний үржигдэхүүнийг 

\(\displaystyle  a+\sqrt{ab}+\sqrt{ac} =\sqrt{a}(\sqrt{a}+\sqrt{b}+\sqrt{c})\leq \sqrt{3a(a+b+c)}\)

 маягаар, хоёрдахь үржигдэхүүнийг нь 

\(\displaystyle  a+\sqrt{ab}+b \leq a+\frac{a+b}2+b\)

 маягаар үнэлснээр бидний батлах гэсэн зүйл батлагдана. \(\Box\)

**Жишээ 6.**  Дурын \({x_1,\ldots,x_m}\) бодит тоонуудын хувьд 

\(\displaystyle  \sum_{n=1}^m \frac{a_n}{\sqrt{n}} \leq \frac{\sqrt\pi}{\sqrt[4]6}\Big( \sum_{n=1}^m |x_n|^{4/3} \Big)^{3/4}\)

 гэж харуул. 

*Бодолт.*  Гёльдерийн тэнцэл бишийг ерөнхий илтгэгчтэйгээр хэрэглэвэл 

\(\displaystyle  \sum_{n=1}^m {x_n}\cdot n^{-1/2} \leq \Big( \sum_{n=1}^m |x_n|^p \Big)^{1/p} \Big( \sum_{n=1}^m j^{-q/2} \Big)^{1/q}\)

 гарна. Одоо 

\(\displaystyle  p=\frac43 \qquad\Longrightarrow\qquad q=\frac1{1-\frac1p}=4\)

 гэдгийг тооцоод, 

\(\displaystyle  \sum_{n=1}^m \frac1{n^2} \leq \sum_{n=1}^\infty \frac1{n^2} = \frac{\pi^2}6\)

 [заагийг](https://en.wikipedia.org/wiki/Basel_problem) ашигласнаар баталгаа дуусна. \(\Box\)
