[Кошийн тэнцэл бишийг](/blog/2020/12/03/amgm/) барууныхан заримдаа «Кошийн 2-р тэнцэл биш» гэж ярьдаг талаар бид өмнө дурдсан. Тэгвэл «Кошийн 1-р тэнцэл биш» гэж байдаг гэсэн үг үү? Тийм. Энэ бол барууны орнуудад «[Коши-Шварцын тэнцэл биш](https://en.wikipedia.org/wiki/Cauchy–Schwarz_inequality)» гэгддэг, манайд *Коши-Буняковскийн тэнцэл биш* гэгддэг алдарт тэнцэл биш юм.

Үүнийг Коши анх нөгөө тэнцэл бишээсээ хэдхэн хуудасны өмнө, «Анализийн курс» номынхоо [хавсралтанд](https://books.google.com/books?id=UrT0KsbDmDwC&pg=PA455) оруулсан. Харин Оросын математикч [Виктор Яковлевич Буняковский](https://en.wikipedia.org/wiki/Viktor_Bunyakovsky) (1804–1889) Кошийн тэнцэл бишийн интеграл бүхий хувилбарыг [1859 онд](http://www-stat.wharton.upenn.edu/~steele/Publications/Books/CSMC/bunyakovsky.pdf), Германы математикч [Герман Амандус Шварц](https://en.wikipedia.org/wiki/Hermann_Schwarz) (1843–1921) давхар интеграл бүхий хувилбарыг нь [1888 онд](http://www-stat.wharton.upenn.edu/~steele/Publications/Books/CSMC/Schwarz.pdf) тус тус баталжээ. Коши-Буняковскийн тэнцэл биш нь математикт хамгийн өргөн хэрэглэгддэг, хамгийн үндсэн, хамгийн чухал тэнцэл бишүүдийн нэг бөгөөд үнэ цэнэ нь Кошийн тэнцэл бишээс ч илүү байж мэдэх юм.

Бид энд зөвхөн интеграл оролцоогүй хувилбарыг нь голчлон авч үзэх тул цэвэр Кошийн нээлтээр хязгаарлагдана гэсэн үг.

**Теорем 1 (Коши-Буняковский).**  Дурын \({x_1,y_1,x_2,y_2,\ldots,x_n,y_n}\) тоонуудын хувьд [

\(\displaystyle  (x_1y_1+\ldots+x_ny_n)^2 \leq (x_1^2+\ldots+x_n^2) (y_1^2+\ldots+y_n^2) \ \ \ \ \ (1)\)

] тэнцэл биш биелэх ба, тэнцэлдээ хүрэх нөхцөл нь 

\(\displaystyle  x_1=\lambda y_1,\qquad x_2=\lambda y_2,\quad\ldots,\quad x_n=\lambda y_n,\)

 байх \({\lambda}\) гэсэн тоо олдох, эсвэл \({y_1=y_2=\ldots=y_n=0}\). Тэнцэлдээ хүрэх нөхцлийг нь бас өөрөөр \({i}\), \({k}\) индекс болгоны хувьд \({x_iy_k=x_ky_i}\) гэж томъёолж болно. 

Энэ теоремын 5 янзын баталгааг дор толилуулна. Хэрэв \({x_1=x_2=\ldots=x_n=0}\) эсвэл \({y_1=y_2=\ldots=y_n=0}\) бол теорем биелэх нь ойлгомжтой. Тэгэхээр бид эдгээр тохиолдлыг баталгаандаа авч үзэхгүй орхиж болно гэдгийг сануулъя. Мөн тэнцэлдээ хүрэх нөхцлийн хоёр томъёоллыг хоорондоо эквивалент гэдгийг харахад амархан. Тухайлбал, \({y_k\neq0}\) гэж үзье. Тэгвэл \({x_iy_k=x_ky_i}\) гэдгээс \({x_i=\frac{x_k}{y_k}y_i}\) гэж гарна. Өөрөөр хэлбэл \({\lambda=\frac{x_k}{y_k}}\). Нөгөө талаас, \({x_i=\lambda y_i}\) ба \({x_k=\lambda y_k}\) гэдгээс \({x_i\lambda y_k=x_k\lambda y_i}\) гэж мөрдөх ба \({\lambda\neq0}\) бол \({x_iy_k=x_ky_i}\) гарна. Харин \({\lambda=0}\) бол \({x_1=\ldots=x_n=0}\) болох тул \({x_iy_k=x_ky_i}\) нөхцөл тэртэй тэргүй биелнэ. Цаашилбал, \({y_1^2+\ldots+y_n^2\neq0}\) бөгөөд [(6)](#ecbs) тэнцэл биш тэнцэлдээ хүрсэн үед \({\lambda}\) параметрийн утга [

\(\displaystyle  \lambda^2=\frac{x_1^2+\ldots+x_n^2}{y_1^2+\ldots+y_n^2} \ \ \ \ \ (2)\)

] нөхцлийг хангана гэдгийг анхаар. 

##  Баталгаа 1: Кошийн тэнцэл биш 

Эхний баталгаа маань Кошийн тэнцэл биш дээр үндэслэгдсэн баталгаа байгаа.

Юуны түрүүнд \({ab\leq\frac{a^2+b^2}2}\) тэнцэл бишийг (үүнийг [Кошийн тэнцэл биш](/blog/2020/12/03/amgm/) гэж бодож ч болно) \({a=x_i/\sqrt{t}}\) ба \({b=y_i\sqrt{t}}\) үед бичвэл [

\(\displaystyle  x_iy_i = \frac{x_i}{\sqrt{t}}\cdot\sqrt{t}y_i \leq \frac1{2t}x_i^2+\frac{t}2y_i^2 \ \ \ \ \ (3)\)

] гарна. Энд \({t>0}\) нь чөлөөтэй параметр. Одоо үүнийгээ \({x_iy_i}\) үржвэр болгоны хувьд хэрэглээд, \({A=\sqrt{x_1^2+\ldots+x_n^2}}\) ба \({B=\sqrt{y_1^2+\ldots+y_n^2}}\) гэсэн товчлол оруулбал 

\(\displaystyle  x_1y_1+\ldots+x_ny_n \leq \frac1{2t}(x_1^2+\ldots+x_n^2)+\frac{t}2(y_1^2+\ldots+y_n^2) = \frac1{2t}A^2+\frac{t}2B^2\)

 болно. Үүний баруун гар талыг хамгийн бага байлгая гэвэл хоёр нэмэгдэхүүнийг хоорондоо тэнцүү байхаар \({t}\) параметрын утгыг сонгож авах хэрэгтэй (Кошийн тэнцэл бишийн бас нэг хэрэглээ): 

\(\displaystyle  \frac1{2t}A^2=\frac{t}2B^2.\)

 Эндээс 

\(\displaystyle  t = \frac{A}{B}\)

 гэж олдох ба буцааж орлуулснаар 

\(\displaystyle  x_1y_1+\ldots+x_ny_n \leq \frac1{2t}A^2+\frac{t}2B^2 = \frac1{2}\cdot\frac{{B}}{{A}}\cdot A^2+\frac12\cdot\frac{{A}}{{B}}\cdot B^2 =AB\)

 болж, тэнцэл биш батлагдана.

Тэнцэл биш тэнцэлдээ хүрэхийн тулд [(3)](#ecbs-pf-1) тэнцэл биш тус бүр тэнцэлдээ хүрэх ёстой. Өөрөөр хэлбэл 

\(\displaystyle  \frac1{2t}x_i=\frac{t}2y_i\)

 нөхцөл биелнэ. Теорем батлагдлаа.

##  Баталгаа 2: Нормчлол 

Удаахь баталгаа өмнөхтэй төстэй боловч эцсийн дүгнэлтээ гаргахдаа параметрчлэл биш, *нормчлол* ашиглана.

Өмнө дурдсанаар \({A=\sqrt{x_1^2+\ldots+x_n^2}=0}\) эсвэл \({B=\sqrt{y_1^2+\ldots+y_n^2}=0}\) үед батлаад байх юм байхгүй учраас \({A\neq0}\) ба \({B\neq0}\) гэе. Ингээд 

\(\displaystyle  a_i=\frac{x_i}{A}, \qquad b_i=\frac{y_i}{B}, \qquad i=1,\ldots,n,\)

 гэсэн шинэ дарааллууд тодорхойлбол эдгээр нь 

\(\displaystyle  a_1^2+\ldots+a_n^2=\frac{x_1^2+\ldots+x_n^2}{A^2}=1\)

 ба 

\(\displaystyle  b_1^2+\ldots+b_n^2=\frac{y_1^2+\ldots+y_n^2}{B^2}=1\)

 чанарыг хангана. Одоо \({ab\leq\frac{a^2+b^2}2}\) тэнцэл бишийг шууд \({a_ib_i}\) үржвэр болгоны хувьд хэрэглэвэл [

\(\displaystyle  a_1b_1+\ldots+a_nb_n \leq \frac{a_1^2+b_1^2}{2}+\frac{a_2^2+b_2^2}2+\ldots+\frac{a_n^2+b_n^2}2 = 1 \ \ \ \ \ (4)\)

] гарах бөгөөд, энэ нь үнэндээ 

\(\displaystyle  a_1b_1+\ldots+a_nb_n=\frac{x_1y_1+\ldots+x_ny_n}{AB}\leq1\)

 гэсэн үг гэдгийг санавал тэнцэл биш батлагдана. Энэ тэнцэл биш тэнцэлдээ хүрсэн нөхцөлд [(4)](#ecbs-pf-2-amgm) томъёонд ашиглагдсан Кошийн тэнцэл биш бүр тэнцэлдээ хүрэх ёстой. Өөрөөр хэлбэл \({a_i=b_i}\) буюу \({{x_i}/A={y_i}/B}\) болж, теорем батлагдлаа.

##  Баталгаа 3: Индукц 

Дараагийн баталгаа маань индукцын баталгаа.

Коши-Буняковскийн тэнцэл бишийг \({n=2}\) үед 

\(\displaystyle  \Delta=(x_1^2+x_2^2)(y_1^2+y_2^2) - (x_1y_1+x_2y_2)^2\geq0\)

 хэлбэртэй бичээд, энэ ялгаврыг задлаад үзвэл 

\(\displaystyle  \begin{array}{rcl}  \Delta &=& x_1^2y_1^2+x_2^2y_1^2+x_1^2y_2^2+x_2^2y_2^2 - (x_1^2y_1^2+x_2^2y_2^2+2x_1y_1x_2y_2)\\ &=& x_2^2y_1^2+x_1^2y_2^2-2x_1y_1x_2y_2\\ &=& (x_2y_1-x_1y_2)^2\geq0, \end{array}\)

 маягаар шууд батлагдана. Үүний тэнцэлдээ хүрэх нөхцөл нь \({x_2y_1=x_1y_2}\).

Дээрх баталгааны явцад 

\(\displaystyle  (x_1^2+x_2^2)(y_1^2+y_2^2) = (x_1y_1+x_2y_2)^2 + (x_2y_1-x_1y_2)^2\)

 гэсэн адилтгал гарч ирснийг тэмдэглэх нь зүйтэй. Энэ бол бидний сайн мэдэх [Диофантын адилтгал](/blog/2019/01/12/diophantus-identity/) юм. Өөрөөр хэлбэл Диофантын адилтгал нь Коши-Буняковскийн тэнцэл бишийн \({n=2}\) тохиолдлыг агуулж байсан гэсэн үг.

Одоо индукц ашиглахын тулд \({A=\sqrt{x_1^2+\ldots+x_n^2}}\) ба \({B=\sqrt{y_1^2+\ldots+y_n^2}}\) товчлолтойгоор 

\(\displaystyle  |x_1y_1+\ldots+x_ny_n| \leq AB\)

 тэнцэл биш биелдэг бөгөөд, тэнцэлдээ хүрэх нөхцөл нь \({1\leq i,k\leq n}\) индекс болгоны хувьд \({x_iy_k=x_ky_i}\) гэж үзье. Тэгвэл 

\(\displaystyle  \begin{array}{rcl}  |x_1y_1+\ldots+x_ny_n+x_{n+1}y_{n+1}| &\leq& |x_1y_1+\ldots+x_ny_n|+|x_{n+1}y_{n+1}|\\ &\leq& AB+|x_{n+1}y_{n+1}|\\ &\leq& \sqrt{A^2+x_{n+1}^2}\sqrt{B^2+y_{n+1}^2} \end{array}\)

 болж, тэнцэл биш батлагдана. Дээрх тэнцэл биш тэнцэлдээ хүрдэг бол 

\(\displaystyle  |x_1y_1+\ldots+x_ny_n|+|x_{n+1}y_{n+1}| = AB+|x_{n+1}y_{n+1}| = \sqrt{A^2+x_{n+1}^2}\sqrt{B^2+y_{n+1}^2}\)

 тул, эхний тэнцэтгэлээс \({|x_1y_1+\ldots+x_ny_n|=AB}\) буюу \({1\leq i,k\leq n}\) индекс болгоны хувьд \({x_iy_k=x_ky_i}\) гэж гарна. Харин сүүлийн тэнцэтгэлээс \({Ay_{n+1}=Bx_{n+1}}\) гэж мөрдөх ба баталгааны ерөнхий чанарыг алдахгүйгээр \({B\neq0}\) гэвэл \({x_{n+1}=\frac{A}{B}y_{n+1}}\). Түүнчлэн, \({x_i=\lambda y_i}\) ба [(2)](#ecbs-lambda) ёсоор \({\lambda=\pm A/B}\). Энд \({\lambda}\)-гийн тэмдгийг яаж ч сонгож авч болох тул эерэг шийдийг нь авснаар \({x_{n+1}=\lambda y_{n+1}}\) болж, теорем батлагдана.

##  Баталгаа 4: Коши 

Дээрх баталгааны \({n=2}\) үед ашиглагдсан Диофантын адилтгалыг ерөнхий тохиолдолд өргөтгөн Коши-Буняковскийг шууд индукцгүйгээр баталж болохгүй юу? Үүнд [Кошийн өөрийнх нь баталгаа](https://books.google.com/books?id=UrT0KsbDmDwC&pg=PA455) хариулт өгнө.

Коши-Буняковскийн тэнцэл бишийн хоёр талын ялгаврыг задлан 

\(\displaystyle  \begin{array}{rcl}  \Delta &=& \big(\sum_i x_i^2\big)\big(\sum y_i^2\big) - \big(\sum_i x_iy_i\big)^2\\ &=& \sum_{i,k} x_i^2y_k^2 - \sum_{i,k} x_iy_ix_ky_k\\ &=& \frac12\sum_{i,k} (x_i^2y_k^2+x_k^2y_i^2) - \sum_{i,k} x_iy_ix_ky_k\\ &=& \frac12\sum_{i,k} (x_i^2y_k^2+x_k^2y_i^2-2x_iy_ky_ix_k)\\ &=& \frac12\sum_{i,k} (x_iy_k-x_ky_i)^2\geq0 \end{array}\)

 маягаар теоремыг шууд баталж болно. Эндээс тэнцэлдээ хүрэх нөхцөл нь ч алган дээр тавьсан мэт илэрхий харагдаж байгаа.

Энэ баталгаанд Диофантын адилтгалыг орлох адилтгал нь 

\(\displaystyle  \big(\sum_i x_i^2\big)\big(\sum y_i^2\big) = \big(\sum_i x_iy_i\big)^2 + \frac12\sum_{i,k} (x_iy_k-x_ky_i)^2\)

 байсан. Үүнийг [Лагранжийн адилтгал](https://en.wikipedia.org/wiki/Lagrange%27s_identity) гэдэг.

##  Баталгаа 5: Шварц 

Эцэст нь, [Шварцын](http://www-stat.wharton.upenn.edu/~steele/Publications/Books/CSMC/Schwarz.pdf) олсон, илүү абстракт чанарын нэг баталгааг сонирхуулъя. Энэ баталгаа нь Коши-Буняковскийн тэнцэл бишийг олон янзаар өргөтгөх боломж олгодог.

Бидэнд \({x=(x_1,\ldots,x_n)\in{\mathbb R}^n}\) ба \({y=(y_1,\ldots,y_n)\in{\mathbb R}^n}\) векторуудын хоорондох *скаляр үржвэр* 

\(\displaystyle  \langle x,y\rangle = x_1y_1+\ldots+x_ny_n\)

 гэсэн ойлголт хэрэг болно. Скаляр үржвэр нь *тэгшхэмтэйгээс* гадна аргумент болгоныхоо хувьд *шугаман* чанартай: 

\(\displaystyle  \langle x,y\rangle = \langle y,x\rangle, \qquad \langle \alpha x+\beta y ,z\rangle = \alpha \langle x,z\rangle + \beta\langle y ,z\rangle .\)

 Түүнчлэн, \({\langle x,x\rangle=0}\) гэдгээс \({x=0}\) гэж мөрдөх нь ойлгомжтой.

**Тодорхойлолт 2.**  Ерөнхий тохиолдолд, бодит шугаман огторгуй \({V}\) дээр тодорхойлогдсон, дараах чанарыг хангадаг \({\langle\cdot,\cdot\rangle:V\times V\rightarrow{\mathbb R}}\) давхар шугаман хэлбэрийг *скаляр үржвэр* (эсвэл *дотоод үржвэр*) гэж нэрлэдэг: 

 
-  \({\langle x,y\rangle=\langle y,x\rangle}\), 
-  \({\langle x,x\rangle\geq0}\), 
-  \({\langle x,x\rangle=0\Longrightarrow x=0}\). 

 

**Теорем 3.**  Бодит шугаман огторгуй \({V}\) дээр \({\langle\cdot,\cdot\rangle:V\times V\rightarrow{\mathbb R}}\) гэсэн скаялар үржвэр өгөгдсөн байг. Тэгвэл [

\(\displaystyle  \langle x,y\rangle\leq\sqrt{\langle x,x\rangle} \sqrt{\langle y,y\rangle} \ \ \ \ \ (5)\)

] тэнцэл биш хүчинтэй ба тэнцэлдээ хүрэх нөхцөл нь \({x=\alpha y}\) эсвэл \({y=0}\). 

*Баталгаа.*  Энд \({y\neq0}\) гэж үзэхэд алдах юм байхгүй. Ингээд \({t\in{\mathbb R}}\) хувьсагчийн дараах квадрат олон гишүүнтийг авч үзье: 

\(\displaystyle  q(t)=\langle x-ty,x-ty\rangle = \langle x,x\rangle -2t\langle x,y\rangle + t^2\langle y,y\rangle .\)

 Скаляр үржвэрийн эерэг чанар ёсоор дурын \({t}\)-ийн хувьд \({q(t)\geq0}\) байх ёстой. Өөрөөр хэлбэл дискриминант нь эерэг биш: 

\(\displaystyle  D = 4\langle x,y\rangle^2 - 4\langle x,x\rangle\langle y,y\rangle \leq0.\)

 Тэнцэл биш батлагдлаа. Энэ тэнцэл биш тэнцэлдээ хүрсэн (ө.х. \({D=0}\)) үед \({q(t)=0}\) тэгшитгэл ганц шийдтэй: \({\langle x-ty,x-ty\rangle=0}\). Эндээс \({x-ty=0}\) гэж гарна. \(\Box\)

**Жишээ 4 (Бодит метрик).**  Хувийн утгууд нь эерэг, тэгшхэмтэй матриц \({A=[a_{jk}]\in{\mathbb R}^{n\times n}}\) өгөгдсөн бол дараах илэрхийлэл скаляр үржвэр тодорхойлно: 

\(\displaystyle  \langle x,y\rangle = x^\top\!Ay=\sum_{jk}a_{jk}x_jy_k.\)

 Тэгэхээр 

\(\displaystyle  x^\top\!Ay\leq \sqrt{x^\top\!Ax}\sqrt{y^\top\!Ay} ,\qquad x,y\in{\mathbb R}^n,\)

 тэнцэл биш хүчинтэй. 

**Жишээ 5 (Буняковский).**  Дараах илэрхийлэл \({[a,b]}\) завсар дээр тодорхойлогдсон тасралтгүй функцүүд дээр скаляр үржвэр тодорхойлно: 

\(\displaystyle  \langle f,g\rangle = \int_a^b f(x)g(x)dx .\)

 Тэгэхээр тасралтгүй функцүүдийн хувьд 

\(\displaystyle  \int_a^b f(x)g(x)dx\leq \Big(\int_a^b |f(x)|^2dx\Big)^{\frac12} \Big(\int_a^b |g(x)|^2dx\Big)^{\frac12}\)

 тэнцэл биш хүчинтэй. 

##  Комплекс өргөтгөл 

Коши-Буняковскийн тэнцэл бишийг комплекс утга авдаг дарааллуудын хувьд өргөтгөж болох уу? Болно. Шварцын баталгааг ингэж өргөтгөхөд скаляр үржвэрийг *Эрмит үржвэрээр* солих хэрэгтэй.

Эрмит үржвэрийн хялбар жишээ \({x=(x_1,\ldots,x_n)\in{\mathbb C}^n}\) ба \({y=(y_1,\ldots,y_n)\in{\mathbb C}^n}\) векторуудын хувьд 

\(\displaystyle  \langle x,y\rangle = x_1\bar y_1+\ldots+x_n\bar y_n\)

 гэж тодорхойлогдоно. Энэ үржвэр дараах чанаруудыг хангана: 

\(\displaystyle  \langle x,y\rangle = \overline{\langle y,x\rangle}, \qquad \langle \alpha x+\beta y ,z\rangle = \alpha \langle x,z\rangle + \beta\langle y ,z\rangle .\)

 Түүнчлэн, \({\langle x,x\rangle=0}\) гэдгээс \({x=0}\) гэж мөрдөх нь ойлгомжтой.

**Тодорхойлолт 6.**  Ерөнхий тохиолдолд, комплекс шугаман огторгуй \({V}\) дээр тодорхойлогдсон, дараах чанарыг хангадаг \({\langle\cdot,\cdot\rangle:V\times V\rightarrow{\mathbb C}}\) буулгалтыг *Эрмит үржвэр* гэж нэрлэдэг: 

 
-  \({\overline{\langle x,y\rangle}=\langle y,x\rangle,\quad x,y\in V}\), 
-  \({\langle \alpha x+\beta y ,z\rangle = \alpha \langle x,z\rangle + \beta\langle y ,z\rangle,\quad x,y\in V,\quad \alpha,\beta\in{\mathbb C}}\), 
-  \({\langle x,x\rangle\geq0}\), 
-  \({\langle x,x\rangle=0\Longrightarrow x=0}\). 

 

**Теорем 7.**  Комплекс шугаман огторгуй \({V}\) дээр \({\langle\cdot,\cdot\rangle:V\times V\rightarrow{\mathbb C}}\) гэсэн Эрмит үржвэр өгөгдсөн байг. Тэгвэл 

\(\displaystyle  |\langle x,y\rangle|\leq\sqrt{\langle x,x\rangle} \sqrt{\langle y,y\rangle}\)

 ба тэнцэлдээ хүрсэн бол ямар нэг комплекс тоо \({\alpha}\)-ийн хувьд \({x=\alpha y}\), эсвэл \({y=0}\). 

*Баталгаа.*  Өмнө дурдсанчлан \({y\neq0}\) гэж үзэхэд алдах юм байхгүй. Ингээд \({t\in{\mathbb R}}\) хувьсагчийн дараах квадрат олон гишүүнтийг авч үзье: 

\(\displaystyle  \begin{array}{rcl}  q(t) &=& \langle x-ty,x-ty\rangle = \langle x,x\rangle - t\langle x,y\rangle - t\langle y,x\rangle + t^2\langle y,y\rangle\\ &=& \langle x,x\rangle - t \big(\langle x,y\rangle + \overline{\langle x,y\rangle}\big) + t^2\langle y,y\rangle\\ &=& \langle x,x\rangle - 2t \Re\langle x,y\rangle + t^2\langle y,y\rangle . \end{array}\)

 Эрмит үржвэрийн эерэг чанар ёсоор дурын \({t}\)-ийн хувьд \({q(t)\geq0}\) байх ёстой. Өөрөөр хэлбэл дискриминант нь эерэг биш: 

\(\displaystyle  D = 4\big(\Re\langle x,y\rangle\big)^2 - 4\langle x,x\rangle\langle y,y\rangle \leq0.\)

 Эндээс 

\(\displaystyle  \Re\langle x,y\rangle\leq\sqrt{\langle x,x\rangle} \sqrt{\langle y,y\rangle}\)

 гарах ба энэ тэнцэл биш тэнцэлдээ хүрсэн үед \({q(t)=0}\) тэгшитгэл ганц шийдтэй: \({\langle x-ty,x-ty\rangle=0}\) буюу \({x=ty}\).

Хэрэв \({x}\)-ийг \({e^{i\theta}}\) гэсэн комплекс тоогоор үржүүлбэл 

\(\displaystyle  \langle e^{i\theta}x,e^{i\theta}x\rangle = e^{i\theta} \langle x,e^{i\theta}x\rangle = e^{i\theta} \overline{\langle e^{i\theta}x,x\rangle} = e^{i\theta} \overline{e^{i\theta}\langle x,x\rangle} = e^{i\theta} e^{-i\theta}\overline{\langle x,x\rangle} = \langle x,x\rangle\)

 тул 

\(\displaystyle  \Re\langle e^{i\theta}x,y\rangle \leq\sqrt{\langle e^{i\theta}x,e^{i\theta}x\rangle} \sqrt{\langle y,y\rangle} =\sqrt{\langle x,x\rangle} \sqrt{\langle y,y\rangle}\)

 гэсэн тэнцэл биш хүчинтэй ба тэнцэлдээ хүрсэн нөхцөлд \({e^{i\theta}x=ty}\). Одоо 

\(\displaystyle  \Re\langle e^{i\theta}x,y\rangle=\Re e^{i\theta}\langle x,y\rangle\)

 гэдгийг санаад, \({\Re e^{i\theta}\langle x,y\rangle=|\langle x,y\rangle|}\) байхаар \({\theta}\) өнцгийг сонгож авснаар 

\(\displaystyle  |\langle x,y\rangle| \leq\sqrt{\langle x,x\rangle} \sqrt{\langle y,y\rangle}\)

 ба тэнцэлдээ хүрсэн нөхцөлд \({x=e^{-i\theta}ty}\) гэж гарна. \(\Box\)

**Жишээ 8 (Эрмит метрик).**  Хувийн утгууд нь эерэг, Эрмит матриц \({A=[a_{jk}]\in{\mathbb C}^{n\times n}}\) өгөгдсөн бол дараах илэрхийлэл дээрх теоремын бүх нөхцлийг хангана (ө.х. Эрмит үржвэр тодорхойлно): 

\(\displaystyle  \langle x,y\rangle = x^\top\!A\bar y=\sum_{jk}a_{jk}x_j\bar y_k.\)

 Тэгэхээр 

\(\displaystyle  |x^\top\!A\bar y| \leq \sqrt{x^\top\!A\bar x}\sqrt{y^\top\!A\bar y} ,\qquad x,y\in{\mathbb C}^n,\)

 тэнцэл биш хүчинтэй. 

**Жишээ 9 (Комплекс Буняковский).**  Дараах илэрхийлэл \({[a,b]}\) завсар дээр тодорхойлогдсон, комплекс утга авдаг тасралтгүй функцүүд дээр Эрмит үржвэр тодорхойлно: 

\(\displaystyle  \langle f,g\rangle = \int_a^b f(x)\bar g(x)dx .\)

 Тэгэхээр тасралтгүй функцүүдийн хувьд 

\(\displaystyle  \Big|\int_a^b f(x)\bar g(x)dx\Big|\leq \Big(\int_a^b |f(x)|^2dx\Big)^{\frac12} \Big(\int_a^b |g(x)|^2dx\Big)^{\frac12}\)

 тэнцэл биш хүчинтэй. 

##  Мөрдлөгөөнүүд 

Коши-Буняковскийн тэнцэл бишийг 

\(\displaystyle  \begin{array}{rcl}  x_1+\ldots+x_n &=& \frac{x_1}{\sqrt{y_1}}\cdot\sqrt{y_1}+\ldots+\frac{x_n}{\sqrt{y_n}}\cdot\sqrt{y_n}\\ &\leq & \Big( \frac{x_1^2}{y_1} + \ldots + \frac{x_n^2}{y_n} \Big)^{\frac12} \big( y_1 + \ldots + y_n \big)^{\frac12} \end{array}\)

 маягаар шууд ашиглан [Шварцын](https://en.wikipedia.org/wiki/Hermann_Schwarz), [Радоны](https://en.wikipedia.org/wiki/Johann_Radon), [Бергстрёмын](https://en.wikipedia.org/wiki/Harald_Bergstrom), [Седракяны](https://en.wikipedia.org/wiki/Nairi_Sedrakyan) тэнцэл биш, мөн [Энгелийн хэлбэр](https://en.wikipedia.org/wiki/Arthur_Engel_(mathematician)), түүнчлэн [Титугийн лемм](https://en.wikipedia.org/wiki/Titu_Andreescu) гэх мэт маш олон нэрээр түгсэн дараах чухал мөрдлөгөөг гаргаж авдаг.

**Мөрдлөгөө 10.**  Дурын \({x_1,\ldots,x_n}\) ба эерэг \({y_1,\ldots,y_n}\) тоонуудын хувьд [

\(\displaystyle  \frac{(x_1+\ldots+x_n)^2}{y_1+\ldots+y_n} \leq \frac{x_1^2}{y_1} + \ldots + \frac{x_n^2}{y_n} \ \ \ \ \ (6)\)

] тэнцэл биш биелэх ба, тэнцэлдээ хүрэх нөхцөл нь 

\(\displaystyle  x_1=\lambda y_1,\qquad x_2=\lambda y_2,\quad\ldots,\quad x_n=\lambda y_n,\)

 байх \({\lambda}\) гэсэн тоо олдох явдал. 

##  Жишээнүүд 

**Жишээ 11.**  Хэрэв \({a,b,c\geq1}\) ба \({\frac1a+\frac1b+\frac1c=2}\) бол 

\(\displaystyle  \sqrt{a-1}+\sqrt{b-1}+\sqrt{c-1}\leq\sqrt{a+b+c}\)

 гэж харуул. 

*Бодолт.*  Коши-Буняковскийн тэнцэл бишийг \({(\sqrt{a},\sqrt{b},\sqrt{c})}\) ба \({(\sqrt{1-\frac1a},\sqrt{1-\frac1b},\sqrt{1-\frac1c})}\) дарааллууд дээр хэрэглэснээр 

\(\displaystyle  \big(\sqrt{a-1}+\sqrt{b-1}+\sqrt{c-1}\big)^2 \leq \big(a+b+c\big) \Big(1-\frac1a+1-\frac1b+1-\frac1c\Big) = a+b+c\)

 болж бодлого бодогдоно. \(\Box\)

**Жишээ 12.**  Бодит эерэг \({x_1,\ldots,x_n}\) тоонууд \({\frac1{x_1}+\ldots+\frac1{x_n}=1}\) нөхцлийг хангадаг бол 

\(\displaystyle  \sum_i\big(x_i+\frac1{x_i}\big)^2\geq\frac{(n^2+1)^2}{n}\)

 байна гэж батал. 

*Бодолт.*  Коши-Буняковскийн тэнцэл бишээс 

\(\displaystyle  \sum_i1\cdot\big(x_i+\frac1{x_i}\big) \leq \Big(\sum_i1^2\Big)^{\frac12} \Big(\sum_i\big(x_i+\frac1{x_i}\big)^2\Big)^{\frac12}\)

 буюу [

\(\displaystyle  \sum_i\big(x_i+\frac1{x_i}\big)^2\geq \frac1n\Big(\sum_i\big(x_i+\frac1{x_i}\big)\Big)^2 \ \ \ \ \ (7)\)

] гэж мөрдөхийг ажиглая. Нөгөө талаас, Коши-Буняковскийн бас нэг хэрэглээ 

\(\displaystyle  n=\sum_i\sqrt{x_i}\cdot\frac1{\sqrt{x_i}} \leq \Big(\sum_ix_i\Big)^{\frac12} \Big(\sum_i\frac1{x_i}\Big)^{\frac12} =\Big(\sum_i\frac1{x_i}\Big)^{\frac12}\)

 буюу 

\(\displaystyle  \sum_i\frac1{x_i}\geq n^2\)

 гэж өгнө. Тэгэхээр 

\(\displaystyle  \sum_i\big(x_i+\frac1{x_i} \big) = 1 + \sum_i\frac1{x_i}\geq1+n^2\)

 болох ба, үүнийгээ [(7)](#ecbs-eg-1) томъёонд орлуулснаар 

\(\displaystyle  \sum_i\big(x_i+\frac1{x_i}\big)^2 \geq \frac1n\Big(\sum_i\big(x_i+\frac1{x_i}\big)\Big)^2 \geq \frac{(n^2+1)^2}{n}\)

 гэж мөрдөнө. Бодлого бодогдлоо. \(\Box\)
