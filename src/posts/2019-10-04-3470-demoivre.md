[Куб тэгшитгэлийг](/blog/2019/06/22/cubic-equations/) бодох дель [Ферро](https://en.wikipedia.org/wiki/Scipione_del_Ferro), [Тарталья](https://en.wikipedia.org/wiki/Niccolo_Fontana_Tartaglia), [Кардано](https://en.wikipedia.org/wiki/Gerolamo_Cardano) нарын томъёо, [4 зэрэгт тэгшитгэлийг](/blog/2019/08/11/quartic-equations/) бодох [Феррарийн ](https://en.wikipedia.org/wiki/Lodovico_Ferrari) томъёотой хамт 1545 онд анх хэвлэгдсэн бол, куб тэгшитгэлийн шийдэгдээгүй үлдсэн «үл задрах» тохиолдлыг нь 1592 онд [Виет](https://en.wikipedia.org/wiki/Francois_Viete) тригонометрийн аргаар шийдсэн билээ. Цаашилбал, 4 зэргийн тэгшитгэлийг бодох шинэ аргыг [Декарт](https://en.wikipedia.org/wiki/René_Descartes) олж, 1637 онд хэвлүүлсэн. Мөн 5 зэргийн тэгшитгэлийг бодох гэсэн [Худде](https://en.wikipedia.org/wiki/Johannes_Hudde), [Грегори](https://en.wikipedia.org/wiki/James_Gregory_(mathematician)), [Чирнхаус](https://en.wikipedia.org/wiki/Ehrenfried_Walther_von_Tschirnhaus), [Лейбниц](https://en.wikipedia.org/wiki/Gottfried_Wilhelm_Leibniz) нарын бүтэлгүй оролдлогуудын тухай [өмнө дурдсан](/blog/2019/09/28/tschirnhaus/). Эдгээр оролдлогууд дундаас нэг чухал үр дүн гарсан нь Чирнхаус өөрийнх нь нэрийг сүүлд зүүсэн хувиргалтыг ашиглан куб тэгшитгэлийг бодсон явдал байв. Чирнхаусын ажил 1683 онд хэвлэгдсэн ч, энэ аргыг 4 зэргийн тэгшитгэл рүү зөвхөн 1771 онд [Лагранж](https://en.wikipedia.org/wiki/Joseph-Louis_Lagrange) өргөтгөснийг бид мэднэ. Лагранжаас өмнө, Чирнхаусаас хойш олон гишүүнт тэгшитгэлийн онолд ахиц гаргасан хүмүүс бол [Колсон](https://en.wikipedia.org/wiki/John_Colson), [Абрахам де Муавр](https://en.wikipedia.org/wiki/Abraham_de_Moivre) (1667–1754), их багш Эйлер, [Этьен Безу](https://en.wikipedia.org/wiki/Étienne_Bézout) (1730–1783) нар юм. Куб тэгшитгэлийн 3 шийдийг Карданы томъёоноос яаж гаргаж авахыг, тухайлбал энэ томъёо нь комплекс тоонууд дээр ч хүчинтэй болохыг анх тайлбарлаж үзүүлсэн [Колсоны](https://en.wikipedia.org/wiki/John_Colson) 1707 оны ажлын тухай бид [хэдийнээ](/blog/2019/06/22/cubic-equations/) дурдсан. Энэ өгүүллээр бид де Муаврын хувь нэмрийн талаар авч үзэх гэж байна.

[ Абрахам де Муавр](https://en.wikipedia.org/wiki/Abraham_de_Moivre) нь Францын [Шампань](https://en.wikipedia.org/wiki/Champagne_(province)) мужид төрсөн ба Парис хотод математикт суралцсаныхаа дараа 1688 онд Англид ирж, үүнээс хойш амьдралынхаа сүүлийн 66 жилийг тэнд өнгөрөөсөн хүн юм. Түүний математикийн мэдлэг чадвар нь тухайн үеийн ихэнх англи профессоруудынхаас хэд дахин илүү байсан ч франц гаралтайн улмаас ажилд орох боломж олдолгүй насан туршдаа хувиараа хүүхдүүдэд давтлага өгч өлөө залгуулдаг байжээ. Тэрбээр маш чухал олон нээлт хийсний нэг нь одоо бидний авч үзэх гэж байгаа \\({n}\\) зэргийн зарим тэгшитгэлийг боддог томъёо бөгөөд үүнтэй холбоотойгоор өөрийн алдарт 

\\(\displaystyle  \cos n\theta + i\sin n\theta = (\cos\theta+i\sin\theta)^n\\)

 адилтгалаа нээсэн түүхтэй.

Де Муавр 1707 онд (Колсоны өгүүлэлтэй нэг дор, нэг сэтгүүлийн дугаарт) хэвлүүлсэн ажлаараа [

\\(\displaystyle  nx+\frac{n(n^2-1)x^3}{3!}+\frac{n(n^2-1)(n^2-3^2)x^5}{5!}+\ldots=a \ \ \ \ \ (1)\\)

] тэгшитгэлийн шийдийг [

\\(\displaystyle  x = {\textstyle\frac12}\sqrt[n]{\sqrt{a^2+1}+a} - {\textstyle\frac12}\sqrt[n]{\sqrt{a^2+1}-a} \ \ \ \ \ (2)\\)

] гэж, харин [

\\(\displaystyle  nx+\frac{n(1-n^2)x^3}{3!}+\frac{n(1-n^2)(3^2-n^2)x^5}{5!}+\ldots=a \ \ \ \ \ (3)\\)

] тэгшитгэлийн шийдийг [

\\(\displaystyle  x = {\textstyle\frac12}\sqrt[n]{a+\sqrt{a^2-1}} + {\textstyle\frac12}\sqrt[n]{a-\sqrt{a^2-1}} \ \ \ \ \ (4)\\)

] гэж харуулсан. Энд \\({n}\\) нь сондгой тоо бол [(1)](#esinh-eq) ба [(3)](#esin-eq) тэгшитгэлүүдийн зүүн гар тал \\({n}\\) зэргийн олон гишүүнт болохыг анзаар. Тухайлбал, \\({n=5}\\) үед [(1)](#esinh-eq) ба [(3)](#esin-eq) нь харгалзан 

\\(\displaystyle  5x+20x^3+16x^5=a, \qquad 5x-20x^3+16x^5=a,\\)

 хэлбэрт орно. Де Муаврын [(2)](#esinh-eq-sol) ба [(4)](#esin-eq-sol) томъёонууд нь \\({n}\\) зэргийн зарим тэгшитгэлийг Карданы томъёотой төстэй томъёогоор бодож болдгийг анхлан үзүүлснээрээ маш чухал үр дүн байсан юм. Жишээ нь, 

\\(\displaystyle  5x+20x^3+16x^5=3,\\)

 тэгшитгэлийн шийд 

\\(\displaystyle  x = {\textstyle\frac12}\sqrt[5]{\sqrt{10}+3} - {\textstyle\frac12}\sqrt[5]{\sqrt{10}-3}\\)

 гэж олдоно.

Дээрх томъёонуудын нууц нь юу вэ гэвэл, \\({x=\sinh t}\\) орлуулгаар [(1)](#esinh-eq) нь [

\\(\displaystyle  \sinh nt = \sinh(n\sinh^{-1}x) = a , \ \ \ \ \ (5)\\)

] харин \\({x=\sin\theta}\\) орлуулгаар [(3)](#esin-eq) нь [

\\(\displaystyle  \sin n\theta = \sin(n\arcsin x) = a , \ \ \ \ \ (6)\\)

] болдог. Өөрөөр хэлбэл, [(1)](#esinh-eq) нь \\({\sinh nt}\\) өгөгдсөн үед \\({\sinh t}\\)-г олох, [(3)](#esin-eq) нь \\({\sin n\theta}\\) өгөгдсөн үед \\({\sin\theta}\\)-г олох бодлого бөгөөд, [(2)](#esinh-eq-sol) ба [(4)](#esin-eq-sol) томъёонууд нь харгалзан [

\\(\displaystyle  x = \sinh t = \sinh({\textstyle\frac1n}\sinh^{-1}a) , \ \ \ \ \ (7)\\)

] ба [

\\(\displaystyle  x = \sin\theta = \sin({\textstyle\frac1n}\arcsin a) , \ \ \ \ \ (8)\\)

] томъёонуудтай эквивалент. Өнцгийг \\({n}\\) тэнцүү хуваах [(3)](#esin-eq) тэгшитгэлийн \\({n=3}\\), \\({n=5}\\), \\({n=45}\\) тохиолдлуудыг бүр 1595 онд Виет, \\({n=3}\\), \\({n=5}\\), \\({n=7}\\) тохиолдлуудыг 1620-иод онд Английн математикч [Генри Бригс](https://en.wikipedia.org/wiki/Henry_Briggs_(mathematician)) (1561–1630) нар гаргаж, синус функцийн хүснэгт ашиглан (ойролцоогоор) бодож байсан гэдгийг тэмдэглэх нь зүйтэй.

 Одоо [(2)](#esinh-eq-sol) ба [(7)](#esinh-eq-sol-1) томъёонуудын хоорондох холбоог сонирхоё. Юуны өмнө, 

\\(\displaystyle  \sinh nt = a , \qquad \cosh nt = \sqrt{\sinh^2\!nt+1} = \sqrt{a^2+1} ,\\)

 тул [(2)](#esinh-eq-sol) томъёог 

\\(\displaystyle  x = \sinh t = {\textstyle\frac12}\sqrt[n]{\cosh nt + \sinh nt} + {\textstyle\frac12}\sqrt[n]{\cosh nt - \sinh nt} .\\)

 хэлбэрт бичиж болно. Харин энэ нь 

\\(\displaystyle  \cosh nt \pm \sinh nt = e^{\pm nt} = (e^{\pm t})^n = (\cosh t \pm \sinh t)^n\\)

 гэсэн адилтгалаас шууд мөрдөж байгаа.

Дээрхтэй төстэйгөөр, 

\\(\displaystyle  \sin n\theta = a , \qquad \cos n\theta = \sqrt{1-\sin^2\!n\theta} = \sqrt{1-a^2} ,\\)

 тул [(4)](#esin-eq-sol) томъёог [

\\(\displaystyle  x = \sin\theta={\textstyle\frac12}\sqrt[n]{\sin\theta+i\cos\theta} + {\textstyle\frac12}\sqrt[n]{\sin\theta-i\cos\theta} \ \ \ \ \ (9)\\)

] хэлбэрт бичиж болно (энд \\({i^2=-1}\\)). Үүнийг де Муаврын алдарт [

\\(\displaystyle  \cos n\theta + i\sin n\theta = (\cos\theta+i\sin\theta)^n \ \ \ \ \ (10)\\)

] адилтгалаас гаргаж болно. Де Муавр өөрийн адилтгалыг яг [(10)](#ede-Moivre) хэлбэрт хэзээ ч бичээгүй боловч, 1707 оныхоо ажилд [(9)](#ede-Moivre-1) хэлбэрийг ашиглан тооцоо хийсэн байдаг ба, 1730 онд 

\\(\displaystyle  \cos\theta={\textstyle\frac12}\sqrt[n]{\cos\theta+i\sin\theta} + {\textstyle\frac12}\sqrt[n]{\cos\theta-i\sin\theta} ,\\)

 адилтгалын гаргалгааг хэвлүүлсэн байдаг.

Эцэст нь, [(1)](#esinh-eq) ба [(3)](#esin-eq) тэнцэтгэлүүд нь харгалзан [(5)](#esinh-eq-1) ба [(6)](#esin-eq-1) тэнцэтгэлүүдтэй яагаад эквивалент юм бэ гэдгийг буюу, [

\\(\displaystyle  \sinh(n\sinh^{-1}x) = nx+\frac{n(n^2-1)x^3}{3!}+\frac{n(n^2-1)(n^2-3^2)x^5}{5!}+\ldots \ \ \ \ \ (11)\\)

] ба [

\\(\displaystyle  \sin(n\arcsin x) = nx+\frac{n(1-n^2)x^3}{3!}+\frac{n(1-n^2)(3^2-n^2)x^5}{5!}+\ldots \ \ \ \ \ (12)\\)

] задаргаанууд яагаад хүчинтэй вэ гэдгийг товч тайлбарлая. Сүүлийнх нь задаргааг Ньютон 1676 онд нэгэн захиандаа дурдсан байдаг тул *Ньютоны олон давхар өнцгийн томъёо* гэх нь бий. Энэ томъёог гаргахын тулд, 

\\(\displaystyle  f(x)=\sin(n\arcsin x)\\)

 гэсэн функц тодорхойлоод, 2 удаа уламжлал авбал 

\\(\displaystyle  f'(x)=\cos(n\arcsin x) \cdot \frac{n}{\sqrt{1-x^2}}\\)

 ба 

\\(\displaystyle  f''(x)=-\sin(n\arcsin x) \cdot \frac{n^2}{1-x^2} + \cos(n\arcsin x) \cdot \frac{nx}{\sqrt{(1-x^2)^3}}\\)

 болно. Ингээд 

\\(\displaystyle  f''(x)=\frac{n^2}{1-x^2}f(x) + \frac{x}{1-x^2}f'(x)\\)

 буюу 

\\(\displaystyle  (1-x^2)f''(x) - xf'(x) + n^2f(x) = 0\\)

 гэсэн дифференциал тэгшитгэлд хүрлээ. Үүнийхээ шийдийг зэрэгт цуваа хэлбэрээр 

\\(\displaystyle  f(x) = \sum_{k=0}^\infty a_kx^k\\)

 гэж эрвэл, коэффициентуудынх нь хооронд 

\\(\displaystyle  a_{k+2} = \frac{k^2-n^2}{(k+1)(k+2)} a_k ,\\)

 гэсэн рекуррент харьцаа гарч ирнэ. Тухайлбал, 

\\(\displaystyle  a_{0} = f(0) = \sin(n\arcsin0) = 0\\)

 тул, тэгш индекстэй бүх коэффициентүүд тэг: 

\\(\displaystyle  a_{2m} = 0 ,\qquad m=0,1,2,\ldots.\\)

 Түүнчлэн 

\\(\displaystyle  a_1 = f'(0)=\cos(n\arcsin 0) \cdot \frac{n}{\sqrt{1-0^2}} = n\\)

 тул, 

\\(\displaystyle  a_{3} = \frac{1^2-n^2}{2\cdot3} n , \qquad a_{5} = \frac{3^2-n^2}{4\cdot5} a_3 = \frac{(3^2-n^2)(1^2-n^2)}{2\cdot3\cdot4\cdot5} n ,\\)

 гэх мэтчилэн, [(12)](#esin-exp) задаргаа гарлаа.

**Дасгал 1.**  Дээрхтэй төстэй аргыг ашиглан [(11)](#esinh-exp) томъёог гарга.
