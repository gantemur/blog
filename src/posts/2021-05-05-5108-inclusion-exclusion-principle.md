[Магадлалыг нэмэх](/blog/2021/05/04/law-of-addition/) \\({P(A\cup B)=P(A)+P(B)}\\) дүрэм нь зөвхөн хоорондоо нийцгүй үзэгдлүүд дээр (ө.х. \\({AB=\emptyset}\\) үед) л хүчинтэй. Энд \\({A\cup B}\\) нь олонлогуудын (буюу үзэгдлүүдийн) нэгдэл, \\({AB=A\cap B}\\) нь огтлолцлыг тэмдэглэж байгааг санаарай. Одоо ерөнхий тохиолдолд юу болохыг сонирхоё.

**Теорем 1.**  Дурын \\({A}\\) ба \\({B}\\) үзэгдлүүдийн хувьд []

\\(\displaystyle  P(A\cup B) = P(A) + P(B) - P(AB) \ \ \ \ \ (1)\\)

][] ба []

\\(\displaystyle  P(A\setminus B) = P(A) - P(AB) \ \ \ \ \ (2)\\)

][] тэнцэтгэлүүд биелнэ. Энд \\({A\setminus B}\\) нь \\({A}\\)-аас \\({B}\\)-г хассан *ялгавар* гэгдэх, \\({A}\\)-д ордог боловч \\({B}\\)-д ордоггүй элементүүдээс бүтэх олонлог:

\\(\displaystyle  A\setminus B = \{x\in A:x\not\in B\} .\\)

*Баталгаа.*  Бид \\({A}\\) үзэгдлийг

\\(\displaystyle  A = (A\setminus B) \cup (AB)\\)

\\(\displaystyle  P(A) = P(A\setminus B) + P(AB)\\)

 болно. Эндээс [(2)](#eset-diff) илэрхий. Үүнтэй төстэйгөөр, \\({A\cup B}\\) үзэгдлийг

\\(\displaystyle  A\cup B = (A\setminus B) \cup (B\setminus A) \cup (AB)\\)

\\(\displaystyle  P(A\cup B) = P(A\setminus B) + P(B\setminus A) + P(AB)\\)

\\(\displaystyle  P(A\setminus B) = P(A) - P(AB), \qquad P(B\setminus A) = P(B) - P(AB),\\)

 гэж бичсэнээр [(1)](#eset-union) томьёо батлагдана. \\(\Box\\)
