Английн математикч [Уильям Генри Юнг](https://en.wikipedia.org/wiki/William_Henry_Young) хэд хэдэн чухал тэнцэл бишийг хүн төрөлхтөнд бэлэглэсний дотор өнгөн дээрээ үүнд нэр өгөх хэрэгтэй юм уу гэмээр хялбар мөртлөө маш өргөн ашиглагддаг нэг тэнцэл биш байдаг. Энэ тэнцэл бишээ тэрбээр [1912 онд](https://royalsocietypublishing.org/doi/10.1098/rspa.1912.0076) томъёолжээ.

**Теорем 1.**  Сөрөг биш \({x,y}\) тоонууд болон \({\frac1p+\frac1q=1}\) байх \({p\geq1}\) ба \({q\geq1}\) рациональ тоонуудын хувьд [

\(\displaystyle  xy\leq \frac{x^p}p+\frac{y^q}q \ \ \ \ \ (1)\)

] тэнцэл биш биелэх ба, тэнцэлдээ хүрэх нөхцөл нь \({x^p=y^q}\). 

Энд \({p}\) параметрыг рациональ байхыг шаарддаг нь бид баталгаандаа анализийн аргуудыг ашиглахгүй гэж зүтгэснээс болсныг дахин сануулъя. Үнэндээ бол Юнгийн тэнцэл биш \({p}\)-ийн иррацаиональ утганд ч биелнэ.

Юнгийн тэнцэл бишид \({\varepsilon>0}\) гэсэн «удирдах» параметр оруулсан [

\(\displaystyle  xy=\varepsilon x\cdot \frac{y}\varepsilon\leq \frac{\varepsilon^px^p}p+\frac{y^q}{q\varepsilon^q} \ \ \ \ \ (2)\)

] хувилбар нь их өргөн тохиолддог. 

##  Баталгаа 

[Өмнөх постоороо](/blog/2020/12/03/amgm/) бид «жинтэй Кошийн тэнцэл биш» гэгддэг үр дүнг гаргаж авсан. Хоёр хувьсагчтай үеийн тухайн тохиолдлыг нь томъёолъё.

**Теорем 2.**  Сөрөг биш \({a,b}\) тоонууд, \({0\leq\lambda\leq1}\) рациональ тооны хувьд [

\(\displaystyle  a^{\lambda}b^{1-\lambda}\leq \lambda a+(1-\lambda)b , \ \ \ \ \ (3)\)

] тэнцэл биш биелэх ба, тэнцэлдээ хүрэх нөхцөл нь \({a=b}\). 

Энэ тэнцэл бишид \({a^\lambda\mapsto x}\) ба \({b^{1-\lambda}\mapsto y}\) гэсэн орлуулга хийснээр бидний батлах гэж зорьсон Юнгийн тэнцэл биш [(1)](#eYoung) мөрдөнө.

##  Жишээнүүд 

**Жишээ 3.**  Сөрөг биш \({x,y}\) тоонууд болон эерэг рациональ \({a,b}\) тоонуудын хувьд 

\(\displaystyle  x^\alpha y^\beta\leq\frac{\alpha}{\alpha+\beta}x^{\alpha+\beta}+\frac{\beta}{\alpha+\beta}y^{\alpha+\beta} ,\)

 тэнцэл биш биелэх ба, тэнцэлдээ хүрэх нөхцөл нь \({x=y}\). 

*Бодолт.*  Жинтэй Кошийн [(3)](#eCauchy-ineq-weight-2) тэнцэл бишийг 

\(\displaystyle  a=x^{\alpha+\beta} , \qquad b=y^{\alpha+\beta} , \qquad \lambda=\frac{\alpha}{\alpha+\beta} ,\)

 тоонуудын хувьд хэрэглэснээр дээрх тэнцэл биш гарч ирнэ. \(\Box\)

**Жишээ 4.**  Сөрөг биш \({x_1,y_1,\ldots,x_n,y_n}\) тоонууд 

\(\displaystyle  x_1^3+\ldots+x_n^3\leq x_1y_1^2+\ldots+ x_ny_n^2\)

 тэнцэл бишийг хангадаг бол 

\(\displaystyle  x_1^3+\ldots+x_n^3 \leq C(y_1^3+\ldots+y_n^3)\)

 байх \({C}\) тогтмол олдоно гэж батал. Энд \({C}\) тогтмол нь \({x_1,y_1,\ldots,x_n,y_n}\) тоонуудаас хамааралгүй байх ёстой. 

*Бодолт.*  Юнгийн тэнцэл бишийн [(2)](#eYoung-eps) хэлбэрийг \({p=3}\) (тэгэхээр \({q=\frac32}\)) илтгэгчтэйгээр хэрэглэвэл дурын \({\varepsilon>0}\) тооны хувьд 

\(\displaystyle  x_iy_i^2\leq \frac{\varepsilon^3x_i^3}3+\frac{2y_i^3}{3\varepsilon^{3/2}}\)

 болох тул 

\(\displaystyle  \sum_i x_i^3 \leq \sum_i x_iy_i^2 \leq \frac{\varepsilon^3}3 \sum_i x_i^3 +\frac{2}{3\varepsilon^{3/2}} \sum_i y_i^3\)

 буюу 

\(\displaystyle  \Big( 1- \frac{\varepsilon^3}3 \Big)\sum_i x_i^3 \leq \frac{2}{3\varepsilon^{3/2}} \sum_i y_i^3\)

 гарна. Одоо жишээ нь \({\varepsilon=\frac12}\) гэж сонгосноор бодлого бодогдоно. Энэ санааг цааш нь хөгжүүлснээр [Гёльдерийн тэнцэл биш](/blog/2020/12/08/holder/) гардаг. \(\Box\)

**Жишээ 5.**  Сөрөг биш \({x,y}\) тоонуудын хувьд 

\(\displaystyle  x^ay^b+x^by^a \leq x^4y+xy^4\)

 тэнцэл биш биелдэг байхын тулд \({a,b}\) илтгэгчид ямар нөхцлийг хангах ёстой вэ? 

*Бодолт.*  Эхлээд юу ч гэсэн тэнцэл бишийг биелдэг гэж үзээд, үүнийгээ батлахын тулд Юнгийн тэнцэл бишийг 

\(\displaystyle  x^ay^b =(x^{a_1}y^{b_1})(x^{a_2}y^{b_2}) \leq \frac{(x^{a_1}y^{b_1})^p}p + \frac{(x^{a_2}y^{b_2})^q}q =\frac{x^4y}p + \frac{xy^4}q\)

 маягаар ашигласан гэж төсөөлье. Тэгвэл дараах нөхцлүүд биелэх ёстой: 

\(\displaystyle  a_1+a_2=a,\quad b_1+b_2=b,\quad a_1p=4,\quad b_1p=1,\quad a_2q=1,\quad b_1q=4 .\)

 Одоо \({\lambda=\frac1p}\) гээд \({\frac1p+\frac1q=1}\) гэдгийг тооцвол 

\(\displaystyle  a_1=4\lambda,\qquad b_1=\lambda,\qquad a_2=1-\lambda,\qquad b_2=4-4\lambda\)

 болох тул 

\(\displaystyle  a = a_1+a_2=1+3\lambda,\qquad b=b_1+b_2=4-3\lambda\)

 гэж гарна. Өөрөөр хэлбэл, \({0<\lambda<1}\) тооны хувьд \({(a,b)}\) хос [

\(\displaystyle  (a,b) = (1+3\lambda,4-3\lambda) \ \ \ \ \ (4)\)

] гэж илэрхийлэгдэж байвал, 

\(\displaystyle  x^ay^b \leq \frac{x^4y}p + \frac{xy^4}q\)

 ба \({x,y}\)-ийн үүргийг сольсноор 

\(\displaystyle  x^by^a \leq \frac{xy^4}p + \frac{x^4y}q\)

 тул, [

\(\displaystyle  x^ay^b+x^by^a \leq x^4y+xy^4 \ \ \ \ \ (5)\)

] тэнцэл биш хүчинтэй. Геометрийн хувьд [(4)](#econv-hull-2d-2pt) нөхцөл нь \({A=(1,4)}\) ба \({B=(4,1)}\) цэгүүдийг холбосон \({AB}\) хэрчим дээр \({(a,b)}\) цэг оршихыг илэрхийлж байгаа. Жишээлбэл, \({(3,2)=(1+2,4-2)}\) тул 

\(\displaystyle  x^3y^2+x^2y^3 \leq x^4y+xy^4.\)

 

[![](/blog/assets/wp-media/2020/12/muirhead2-1.png)](/blog/assets/wp-media/2020/12/muirhead2-1.png)

 

Одоо \({(a,b)}\) цэг \({AB}\) хэрчмийн гадна байх тохиолдолд юу болохыг сонирхоё. Хэрэв \({a+b\neq5}\) бол, \({x=y}\) гэж авснаар [(5)](#epf-eg-muirhead-2) тэнцэл бишийн зүүн гар тал \({2x^{a+b}}\), харин баруун гар тал нь \({2x^5}\) болох тул, тэнцэл бишийг зөрчихөд амархан. Эцэст нь, \({(a,b)}\) цэг \({A}\) ба \({B}\) цэгүүдийг холбосон *шулуун* дээр орших боловч, \({AB}\) хэрчмийн гадна байх тохиолдлыг авч үзье. Тодруулбал \({a+b=5}\) боловч \({a<1}\) эсвэл \({a>4}\) байна гэсэн үг. Энэ тохиолдолд \({y=1}\) гэж орлуулснаар [(5)](#epf-eg-muirhead-2) тэнцэл биш 

\(\displaystyle  x^a+x^{5-a} \leq x^4+x\)

 болох ба жишээ нь \({x}\) их үед (\({a>4}\) эсвэл \({5-a>4}\) гэдгийг санавал) зүүн гар тал нь баруун гар талаасаа их болох нь ойлгомжтой.

Тэгэхээр [(5)](#epf-eg-muirhead-2) тэнцэл биш биелэх зайлшгүй бөгөөд хүрэлцээтэй нөхцөл нь \({(a,b)\in AB}\) юм. Дээрх санааг цааш нь хөгжүүлснээр *Мюрхедийн тэнцэл биш* гардаг. \(\Box\)
